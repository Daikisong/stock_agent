# Round 8 Empty Archetype Backfill

Round 8은 비어 있거나 얇은 archetype을 성공/반례/가격경로까지 채우기 위한 calibration 문서다.

쉬운 예시:

```text
건설주가 유동성 지원 뉴스로 반등했다
하지만 PF, 미분양, cash flow가 해결되지 않았다
= 구조적 E2R 성공이 아니라 credit_relief_rally 반례
```

## 추가 판정값

`price_pattern`에는 다음 값을 명시적으로 쓴다.

- `straight_rerating`
- `stair_step_rerating`
- `cycle_boom_bust`
- `theme_overheat`
- `accounting_trust_break`
- `event_premium`
- `credit_relief_rally`
- `reopening_cycle`
- `policy_contract_delay`

## Round 8 Backfill Targets

- Turnaround: OPM/FCF 개선이 지속되는지, 일회성 비용절감이 아닌지 검증
- Commodity: refining, chemical, steel/metal, rare metals를 분리
- Auto: 완성차 shareholder return과 부품 고객다변화를 분리
- Financial/Value-Up: ROE-PBR-CET1-주주환원 실행 검증
- CDMO: pre-revenue biotech과 분리해 장기계약, capacity, 가동률, FCF 검증
- Royalty biotech: approval -> commercialization -> royalty/revenue 경로 검증
- Platform: MAU/AI narrative가 아니라 ARPU/OPM/FCF 검증
- Shipping: cycle boom-bust와 구조적 Green을 분리
- Nuclear: policy headline과 legal/project delay를 분리
- AI Data Center: order/revenue/capacity bottleneck 확인
- Memory/HBM: Stage 3 성공과 4B-watch를 동시에 calibration

## What Not To Change

- production scoring에 점수비중을 적용하지 않는다.
- Stage 3-Green threshold를 낮추지 않는다.
- 케이스 이름을 scoring 조건으로 쓰지 않는다.
- event premium, reopening cycle, credit relief rally를 true rerating으로 부르지 않는다.
