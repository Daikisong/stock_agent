방금 이 공정으로 정확히 한거야? 
더 자세하게 어디서 어떤걸 검색했고 등등 예시도 덧붙여서 있었으면 좋겠어
정확하게 너가 했던 방식대로 구축해야하니깐 예시가 많이 중요해 레퍼런스삼아야하니깐

응. **공정의 논리는 방금 말한 E2R 파이프라인 그대로 썼다.**
다만 아주 정확히 말하면, 내가 방금 실제로 한 건 **완전 자동화된 에이전트 실행**이 아니라, 그 에이전트가 해야 할 일을 **수동/반자동 웹 딥서치로 모사한 것**이야.

즉 이런 차이야.

| 구분     | 방금 내가 실제로 한 방식                        | 앞으로 구축할 자동 에이전트 방식            |
| ------ | ------------------------------------- | ----------------------------- |
| 유니버스   | 전력기기·변압기 관련 주요 종목을 수동 지정              | 코스피·코스닥 전 종목 자동 스캔            |
| 주가     | 리포트의 당시 현재가·52주 고저 + 공개 주가 페이지로 수동 확인 | KRX/KIS API로 일별 가격 자동 적재      |
| 재무/EPS | 리포트 PDF의 추정 EPS·PER·PBR·OPM 수동 추출     | FnGuide/QuantiWise 컨센서스 자동 적재 |
| 공시     | 리포트가 인용한 신규시설투자·수주 내용을 수동 확인          | OpenDART/KIND 공시 자동 수집        |
| 뉴스/산업  | 글로벌 전력망·데이터센터·변압기 쇼티지 뉴스 수동 확인        | 뉴스 API·리포트 키워드로 섹터 레짐 점수화     |
| 단계 분류  | 내가 직접 Stage 1~4로 판정                   | 점수식과 LLM 검증으로 자동 분류           |

그래서 구축할 때는 **내가 수동으로 검색한 행위를 그대로 코드화**하면 된다.

# 1. 방금 실제로 한 조사 방식

내가 방금 한 흐름은 이거야.

```text
1. 변압기/전력기기 주요 종목 선정
   HD현대일렉트릭, 효성중공업, 일진전기, LS ELECTRIC, 산일전기, 제룡전기, 대한전선

2. 각 종목별 과거 리포트 검색
   “언제 처음 숫자와 리레이팅 논리가 동시에 나왔는가?”를 찾음

3. 리포트 PDF 열람
   현재주가, 52주 고저, 목표주가, EPS, PER, PBR, OPM, ROE, 수주잔고, CAPA, 수출비중 추출

4. 공시·뉴스성 이벤트 확인
   신규시설투자, 수주잔고, 미국향 매출, 데이터센터향 수요, 고마진 프로젝트 등 확인

5. 가격 단계 계산
   당시 현재가가 52주 저점 대비 얼마나 올랐는지 계산

6. Stage 1~4 판정
   레이더 → 후보 편입 → 고확신 → 졸업/피크아웃 감시로 분류

7. 현재 또는 이후 주가와 비교
   그 인식 시점 이후 얼마나 더 상승했는지 확인
```

핵심은 “검색해서 좋은 뉴스 찾기”가 아니라, **그 시점에 이미 공개되어 있던 숫자가 E2R 조건을 충족했는지**를 본 거야.

# 3. 실제로 내가 뽑은 문서와 추출값

## 3-1. 효성중공업

내가 찾은 문서는 2023년 5월 2일 이베스트투자증권 리포트였어. 제목 자체가 **“저마진 수주 정리 1Q 거의 완료. 갈수록 좋다”**였고, 1Q23 실적은 부진했지만 그 이유가 중공업 부문의 저마진·역마진 수주 정리였다고 설명했다. 이건 “현재 실적은 더럽지만 다음 이익 체력이 좋아질 수 있다”는 전형적인 E2R 초입 신호야. ([Naver][1])

내가 추출한 값은 이거야.

| 항목        |            리포트상 값 |
| --------- | ----------------: |
| 리포트일      |        2023.05.02 |
| 현재주가      |           78,500원 |
| 52주 최고/최저 | 89,800원 / 49,050원 |
| 2023E EPS |            6,188원 |
| 2024E EPS |           13,368원 |
| 2025E EPS |           18,739원 |
| 2024E PER |              5.9배 |
| 2025E PER |              4.2배 |
| 2024E PBR |              0.5배 |
| 2025E PBR |              0.4배 |

여기서 에이전트가 봤을 신호는 명확해.

```text
현재 실적 부진
+ 부진 원인이 저마진 수주 정리
+ 다음 EPS가 크게 증가
+ forward PER/PBR이 매우 낮음
+ 전력기기 업황 호황 논리 존재
= Stage 2 후보 편입
```

효성중공업은 **Stage 2**로 잡혔어야 해.
아직 3차 고확신은 아니야. 왜냐하면 그 시점에는 “다음 분기부터 좋아질 것”이라는 예측 성격이 강했고, 연속된 실적 서프라이즈와 멀티플 상향이 충분히 확인된 뒤는 아니었기 때문이야.

## 3-2. HD현대일렉트릭 2023년 7월 5일

여기서 내가 찾은 리포트는 **“단기 변동성 우려? 갈 길이 아직 먼데 무엇이 두려우랴!”**였어. 이 리포트가 중요했던 이유는 이미 주가가 많이 올랐는데도, 리포트가 실적 전망치와 멀티플을 동시에 상향했다는 점이야. 당시 2Q23 영업이익을 전년 대비 112% 증가로 전망했고, 수주가 매출화되는 리드타임이 길어지면서 2024년 이후 실적 전망치 상향 요인이 된다고 봤다. 목표주가도 67,000원에서 89,000원으로 33% 상향했다. ([Naver][2])

추출값은 이거야.

| 항목             |            리포트상 값 |
| -------------- | ----------------: |
| 리포트일           |        2023.07.05 |
| 현재주가           |           67,300원 |
| 목표주가           |           89,000원 |
| 52주 최고/최저      | 67,300원 / 22,050원 |
| 2Q23E 매출       |           6,930억원 |
| 2Q23E 영업이익     |             577억원 |
| 2Q23E 영업이익 YoY |             +112% |
| 목표주가 상향률       |              +33% |
| 12개월 상대수익률     |           +161.4% |

여기서 굉장히 중요한 포인트가 있어.

**당시 주가는 이미 52주 저점 대비 약 3배였다.**
보통 사람은 “너무 올랐다”고 보지만, E2R 에이전트는 이렇게 봐야 해.

```text
주가가 이미 많이 올랐음
하지만:
- 수주잔고 증가
- 장기 물량 수주 증가
- 실적 전망치 상향
- Multiple 상향
- forward 실적이 더 좋아짐
= 아직 졸업생이 아니라 Stage 2~3 진행형
```

이 시점은 **Stage 2 상단 또는 Stage 3 진입 전야**야.

## 3-3. HD현대일렉트릭 2023년 7월 27일

2023년 7월 27일 신한투자증권 리포트 **“줄을 서시오”**는 Stage 3 고확신 신호에 가까웠다. 첫 페이지부터 “2Q23 Re 눈에 띄는 이익률 증가”, “끝없이 쌓이는 수주”, “목표주가 21% 상향”이 동시에 나온다. 현재주가는 69,600원, 목표주가는 87,000원이었다. 

추출값은 이거야.

| 항목        |                    리포트상 값 |
| --------- | ------------------------: |
| 리포트일      |                2023.07.27 |
| 현재주가      |                   69,600원 |
| 목표주가      |                   87,000원 |
| 목표주가 변화   |                        상향 |
| 상승여력      |                     25.0% |
| 52주 최고/최저 |         80,600원 / 24,000원 |
| 핵심 문구     | 이익률 증가, 수주 증가, 목표가 21% 상향 |

이건 3차 고확신이야.

```text
실적 확인
+ 수주잔고 확인
+ 목표주가 상향
+ 시장이 적용 밸류를 다시 올림
= Stage 3 고확신
```

자동화에서는 이 조건을 이렇게 코딩하면 된다.

```python
if report.has_actual_result_review \
   and report.mentions("이익률 증가") \
   and report.mentions("수주") \
   and report.target_price_revision_pct >= 20:
    stage = 3
```

## 3-4. 일진전기

일진전기는 2023년 9월 신규시설투자 공시가 1차 레이더였고, 2024년 3월 27일 SK증권 리포트가 2차 후보 편입 신호였어. 리포트는 초고압변압기 수요 증가에 대응하기 위해 680억원 규모 신규 공장을 건설하고, 변압기 CAPA가 2,600억원에서 4,330억원으로 늘어날 것으로 봤다. 현재주가는 19,300원, 52주 최저가는 5,767원이었다. 

추출값은 이거야.

| 항목        |                 값 |
| --------- | ----------------: |
| 리포트일      |        2024.03.27 |
| 현재주가      |           19,300원 |
| 목표주가      |           25,000원 |
| 52주 최고/최저 |  19,300원 / 5,767원 |
| 신규시설투자    |             680억원 |
| 변압기 CAPA  | 2,600억원 → 4,330억원 |
| 전선 CAPA   | 3,800억원 → 6,200억원 |
| 2025E PER |             13.1배 |

여기서 판단은 이거야.

```text
이미 신고가

+ 초고압변압기 수요 대응
+ forward PER 11~13배
+ 대장주 후순환 가능성
= Stage 2 후보 편입
```

이건 Stage 3까지는 아니고 Stage 2야.
왜냐하면 CAPA 증설은 강한 신호지만, 아직 증설 효과가 실제 매출·영업이익으로 충분히 확인된 시점은 아니었기 때문이야.

## 3-5. LS ELECTRIC

LS ELECTRIC은 2024년 4월 30일 리포트가 핵심이었어. 제목은 **“일시적 서프, 업황은 장기 호황 기대”**였고, 현재주가는 173,300원, 목표주가는 200,000원으로 상향됐다. 리포트는 “1Q24 Re 국내 전력기기가 이끈 서프라이즈”, “목표주가 27% 상향”을 제시했다. 

추출값은 이거야.

| 항목        |                    값 |
| --------- | -------------------: |
| 리포트일      |           2024.04.30 |

| 목표주가      |             200,000원 |

실적 서프라이즈와 목표가 상향이 있으니 Stage 3처럼 보이지만, 내가 A-/A로 본 이유는 **EPS 폭발성 대비 주가·멀티플 반영 속도가 빨랐기 때문**이야.

자동화에서는 이렇게 처리해야 한다.

if result_surprise and target_price_up:
    stage_candidate = 3

if valuation_score < threshold or price_12m_return_too_high:
    stage = min(stage_candidate, 2.5)
```

즉 LS ELECTRIC은 “좋지만, HD현대일렉트릭·효성중공업보다 조금 늦게 잡힌 리레이팅 종목”으로 분류해야 해.

역대 최대 실적
+ OPM 36.1%
+ 수출 비중 98.1%
+ 미국향 79.5%
+ 수주잔고 증가
+ 목표주가 상향
= Stage 3 고확신
```

다만 신규 발굴 초입은 아니야.
“이미 많이 올랐지만 실적이 같이 따라온 고확신 리레이팅주”로 봐야 해.

---

## 3-7. 제룡전기

제룡전기는 2024년 3월 8일 유안타증권 리포트에서 Stage 2 후보로 잡혔을 가능성이 높아. 당시 리포트는 제룡전기를 배전 변압기 100% 업체로 설명했고, 수출 비중이 2021년 25%에서 2023년 80%로 급증했으며, 2023년 잠정 영업이익 702억원, OPM 38.1%, 수주잔고 3,231억원을 제시했다. ([Naver][3])

그런데 2026년에 확인한 사업보고서 쪽에서는 2025년 매출이 전년 대비 14.7% 감소하고, 영업이익이 31.5% 감소했다는 내용이 나온다. 그래서 제룡전기는 “2024년에는 좋은 E2R 후보였지만, 이후 피크아웃 감시로 내려가야 하는 사례”로 분류했다. ([KIND][4])

판정 로직은 이거야.

```text
2024년 3월:
수출 급증 + OPM 38.1% + 수주잔고 증가
= Stage 2 후보

2025년 이후:
매출 감소 + 영업이익 감소
= Stage 4 피크아웃 감시
```

이게 자동화에서 굉장히 중요하다.
좋은 후보를 찾는 것만큼, **좋던 후보가 꺾이는 신호를 잡는 것**이 중요해.

---

## 3-8. 대한전선

대한전선은 내가 일부러 “E2R 본류가 아니라 모멘텀형”으로 분류한 사례야. 2026년 4월 14일 유안타증권 리포트는 1Q26 매출 1조204억원, 영업이익 447억원, OPM 4.4%로 컨센서스 상회를 전망했다. 현재주가는 32,000원, 52주 고저는 36,000원/11,260원, 12개월 절대수익률은 +183.7%였다. 

그런데 같은 리포트의 밸류 표를 보면 2026F PER 83.8배, 2027F PER 62.5배, 2026F ROE 4.3%였다. 이건 우리가 원하는 “EPS 폭발 + 아직 싼 밸류”가 아니야. 

판정은 이거야.

```text
좋은 점:
- 고수익 프로젝트 비중 확대
- 컨센서스 상회 가능
- 수주잔고 보유
- 전력망 테마 강함

나쁜 점:
- 12개월 +183.7% 상승
- 2026F PER 83.8배
- 2026F ROE 4.3%
- EPS보다 테마/멀티플이 먼저 감

= Stage 2/3 E2R이 아니라 모멘텀형 B/C
```

이걸 구분하는 게 중요해.
돈이 되는 트레이딩 종목과 **진짜 EPS-first 리레이팅 종목**은 다르다.

# 4. 실제 Stage 분류 기준

이제 방금 내가 쓴 1차·2차·3차·4차 기준을 더 정확히 코드화하면 이거야.

---

## Stage 1: 레이더 진입

**뜻:** 아직 매수 후보는 아니지만, 산업 또는 기업에서 변화의 냄새가 난다.

**소스:**

* 뉴스
* 산업 리포트
* 신규시설투자 공시
* 단일판매·공급계약 공시
* 증권사 첫 언급 리포트
* 섹터 상대강도 상승

**조건 예시:**

```text
Theme_Regime_Score ≥ 70
또는
Company_Event_Score ≥ 60
```

**트리거 키워드:**

```text
신규시설투자
CAPA 증설
수주잔고 증가
공급 부족
리드타임 증가
ASP 상승
북미 수출
미국향 매출
고마진 프로젝트
데이터센터 전력
전력망 교체
```

**실제 예시:**

일진전기의 2023년 9월 신규시설투자 공시는 Stage 1이다.
아직 실적이 폭발한 건 아니지만, 초고압변압기 수요 증가에 대응해 680억원 규모 신규 공장을 짓는다는 사실이 나왔다. 이건 “수요가 진짜라서 회사가 돈을 쓴다”는 신호다. 

---

## Stage 2: 후보 편입

**뜻:** 숫자와 스토리가 같이 보이기 시작한다.

**조건 예시:**

```text
E2R_Total_Score ≥ 65
EPS_Explosion_Score ≥ 60
Valuation_Score ≥ 50
Rerating_Thesis_Score ≥ 60
```

**정량 기준:**

```text
FY2 영업이익 ≥ TTM 영업이익 × 2
또는 적자 → 흑자 전환

OP 증가분 / 시총 ≥ 10%

OPM 개선 ≥ 2%p

Forward PER이 업종 평균 대비 부담스럽지 않음

PBR 대비 예상 ROE가 매력적

수주잔고 / 매출 ≥ 0.8배

수주, CAPA, ASP, 고부가 제품 중 하나 이상 확인
```

**실제 예시:**

효성중공업 2023년 5월 2일이 Stage 2다.
현재 실적은 부진했지만, 저마진 수주 정리 후 이익률 회복 가능성이 있었고, 2024E·2025E EPS가 크게 증가하며 forward PER/PBR이 매우 낮았다. ([Naver][1])

일진전기 2024년 3월 27일도 Stage 2다.
CAPA 증설, 초고압변압기 수요, forward PER 11~13배, 대장주 후순환 가능성이 동시에 보였다. 

---

## Stage 3: 고확신 편입

**뜻:** 실적, 수주, 컨센서스, 목표주가, 리레이팅 서사가 같이 움직인다.

**조건 예시:**

```text
E2R_Total_Score ≥ 80
EPS_Explosion_Score ≥ 75
Revision_Score ≥ 65
Rerating_Thesis_Score ≥ 75
Risk_Penalty 낮음
```

**강한 트리거:**

```text
실적 서프라이즈
영업이익 컨센서스 상회
목표주가 20% 이상 상향
적용 PER/PBR 상향
수주잔고 사상 최대
OPM 전년 대비 +3%p 이상 개선
수출 비중 급증
미국향·북미향 매출 증가
고마진 프로젝트 매출 인식
```

**실제 예시:**

HD현대일렉트릭 2023년 7월 27일은 Stage 3다.
“2Q23 이익률 증가”, “끝없이 쌓이는 수주”, “목표주가 21% 상향”이 한 번에 나왔다. 

산일전기 2025년 8월 7일도 Stage 3다.
OPM 36.1%, 수출 비중 98.1%, 미국향 79.5%, 수주잔고 4,195억원, 역대 최대 실적이 동시에 나왔다. 

---

## Stage 4: 졸업생 또는 피크아웃 감시

**뜻:** 좋은 기업이지만 발굴 초입은 아니다. 또는 이익 모멘텀이 꺾이기 시작한다.

**졸업생 조건:**

```text
12개월 상승률 ≥ 300%
그리고
Forward PER ≥ 35~50배
또는
PBR ≥ 8~10배
그리고
EPS 상향 속도 둔화
```

**피크아웃 조건:**

```text
매출 성장률 둔화
영업이익률 하락
신규수주 감소
수주잔고 감소
컨센서스 하향
재고 증가
매출채권 증가
관세/경쟁/가격 하락 언급 증가
```

**실제 예시:**

제룡전기는 2024년에는 좋은 후보였지만, 2025년 실적에서 매출과 영업이익이 감소하면서 Stage 4 피크아웃 감시로 내려가야 했다. ([KIND][4])

대한전선은 전력망 모멘텀은 강하지만, 2026F PER 83.8배와 낮은 ROE 때문에 E2R S급이 아니라 모멘텀형으로 분류해야 했다.

# 7. 점수식

방금 내가 머릿속으로 한 판정을 숫자로 바꾸면 이거야.

```text
E2R_Total_Score =
  0.25 × EPS_Explosion_Score
+ 0.18 × Valuation_Score
+ 0.15 × Revision_Score
+ 0.15 × Rerating_Thesis_Score
+ 0.10 × Order_Backlog_or_CAPA_Score
+ 0.07 × Price_Stage_Score
+ 0.05 × Quality_Score
+ 0.05 × Theme_Regime_Score
- Risk_Penalty
```

## EPS Explosion Score

```text
EPS_Explosion_Score =
  0.30 × OP_growth_score
+ 0.20 × EPS_growth_score
+ 0.20 × OP_delta_to_mcap_score
+ 0.15 × OPM_expansion_score
+ 0.10 × ROE_expansion_score
+ 0.05 × turnaround_bonus
```

강한 점수 조건:

```text
FY2 OP / TTM OP ≥ 3배
FY2 EPS / TTM EPS ≥ 3배
OP 증가분 / 시총 ≥ 10~20%
OPM +3~5%p 이상 개선
적자 → 흑자 전환
```

## Valuation Score

```text
Valuation_Score =
  0.30 × forward_PER_attractiveness
+ 0.25 × ROE_to_PBR
+ 0.20 × OP_delta_to_mcap
+ 0.15 × peer_discount
+ 0.10 × PEG_inverse
```

예를 들어 효성중공업 2023년 5월은 2024E PER 5.9배, 2025E PER 4.2배, PBR 0.5~0.4배였기 때문에 Valuation Score가 매우 높아야 한다. 

반대로 대한전선은 2026F PER 83.8배, ROE 4.3%였으므로 Valuation Score가 낮아야 한다. 

## Revision Score

```text
Revision_Score =
  0.30 × EPS_revision_1m
+ 0.25 × OP_revision_1m
+ 0.20 × EPS_revision_3m
+ 0.15 × target_price_revision_1m
+ 0.10 × analyst_count_increase
```

리포트에서 목표주가가 20~30% 상향되면 이 점수가 올라간다.
HD현대일렉트릭 2023년 7월 5일은 목표주가 33% 상향, 7월 27일은 목표주가 21% 상향이라 Revision Score가 강하게 올라가야 한다. ([Naver][2])

## Price Stage Score

```text
if price_6m_return < 30% and eps_revision_3m > 20%:
    score = 100

elif 30 <= price_6m_return < 100 and eps_revision_3m > 20:
    score = 85

elif 100 <= price_6m_return < 250 and eps_revision_3m > 30:
    score = 70

elif price_12m_return > 300 and valuation_still_cheap:
    score = 55

elif price_12m_return > 300 and valuation_expensive:
    score = 25

elif price_up_but_eps_flat:
    score = 10
```

여기서 핵심은 **이미 오른 종목을 자동 탈락시키면 안 된다**는 것.
HD현대일렉트릭은 2023년 7월 5일 이미 52주 저점 22,050원에서 현재가 67,300원까지 올라 있었지만, 그때도 실적 전망과 멀티플 상향이 남아 있었다. ([Naver][2])

# 10. 최종적으로 네가 구축해야 할 핵심

방금 내가 한 방식의 본질은 이거야.

```text
리포트 검색
→ 숫자 추출
→ 당시 가격 추출
→ EPS·OP 증가 가능성 확인
→ 밸류 부담 확인
→ 수주/CAPA/ASP/OPM 확인
→ 시장이 기존 업종에서 새 병목 기업으로 재분류하는지 판단
→ Stage 1~4 분류
```

자동화는 이걸 그대로 반복하면 된다.

가장 중요한 건 이 문장이다.

> **주가가 오른 종목을 찾는 게 아니라,
> 주가가 오른 이유보다 EPS와 리레이팅 여지가 더 크게 남아 있는 종목을 찾는 것.**

효성중공업은 2023년 5월에 forward PER/PBR이 너무 싸서 잡혔어야 했고, HD현대일렉트릭은 2023년 7월 이미 많이 올랐지만 실적 전망치와 멀티플이 같이 상향되면서 잡혔어야 했고, 일진전기는 CAPA 증설과 forward valuation으로 후순환 후보에 들어왔어야 했고, 산일전기는 2025년 고마진·수출·수주잔고가 숫자로 증명되며 고확신으로 올라갔어야 했다.

반대로 대한전선은 전력망 모멘텀은 강하지만 PER·ROE 구조가 E2R S급에는 맞지 않았고, 제룡전기는 2024년에는 좋았지만 2025년 피크아웃 신호가 나오며 등급을 내려야 했다.
