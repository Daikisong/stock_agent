# Round 15 Theme Absorption v0.5

Round 15 absorbs additional theme families that were still too thin after Round 14.

The rule stays the same:

```text
theme tag -> large sector -> archetype -> evidence field -> score hypothesis -> price-path validation
```

Theme names help search and routing. They do not create score.

## Added Theme Families

- Waste / recycling / waste-to-energy
- Refining oil spread
- Chemical spread
- Steel / metal spread
- Retail / e-commerce / logistics
- Insurance / financial value-up
- Digital asset / tokenization / payment
- Hydrogen / renewable policy
- Battery recycling / ESS shift
- Tire / auto component spread
- Diagnostics / infectious event demand
- Speculative science themes
- Agri / livestock / food commodity cycles

## Easy Examples

`폐기물처리` can be close to infrastructure if the company has permits, facilities, long-term contracts, utilization, and repeat FCF.

`화학` can look cheap after a loss cycle, but China oversupply can still make it a 4C risk. Cheap valuation alone is not E2R.

`전염병 진단` can show explosive EPS in a pandemic, but if the demand is one-off, it should be Red/Yellow or 4B-watch, not Stage 3-Green.

## What Did Not Change

- Production scoring did not change.
- StageClassifier thresholds did not change.
- Case/theme labels are not candidate-generation inputs.
- Stage 3-Green remains strict and evidence-backed.
