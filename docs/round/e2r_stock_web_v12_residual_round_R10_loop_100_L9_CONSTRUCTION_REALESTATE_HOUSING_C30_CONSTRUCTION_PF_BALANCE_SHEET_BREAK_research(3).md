## 8. Residual finding

C30 residual error is not “construction sector support was good/bad.” The decisive split is:

```text
Positive C30 path:
  policy/liquidity support
  + issuer-level backlog or public/overseas pipeline
  + refinancing path or lower PF stress
  + margin/FCF bridge
  => Stage2 or Stage2-Actionable can be justified.

Counterexample C30 path:
  policy/liquidity headline only
  + low PBR or sector rebound label
  + unresolved PF guarantee / refinancing / working-capital risk
  + weak margin bridge
  => do not upgrade to Actionable; route to Stage1-Watch or local Stage4B watch.
```

The current calibrated profile already blocks pure price-only blowoff, but this loop shows a narrower C30 weakness: policy support can still be over-credited if balance-sheet/PF-refinancing fields are not explicit.
