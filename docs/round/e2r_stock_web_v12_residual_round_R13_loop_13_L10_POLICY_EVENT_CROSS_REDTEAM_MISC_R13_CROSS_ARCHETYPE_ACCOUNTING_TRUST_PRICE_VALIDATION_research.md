## 17. Sector-Specific Rule Candidate

No new sector-specific production rule is proposed. R13 is intentionally cross-archetype. The rule belongs above individual sectors as a canonical guardrail: each sector can express the same mechanism through its own evidence language.

## 18. Canonical-Archetype Rule Candidate

```text
rule_scope = canonical_archetype_specific / cross_archetype_shadow_guard
rule_name = accounting_trust_verifiability_gate
candidate_behavior:
  if Stage2/Yellow/Green score depends mainly on event, policy, theme beta, or relative strength:
      require at least one direct verifiability bridge:
        - cashflow / margin bridge
        - contract close / tender cap / financing close
        - channel reorder / inventory normalization
        - customer qualification or call-off risk cleared
        - balance-sheet / PF repair evidence
        - confirmed revision or capital-return execution
      if absent:
          cap at Stage2-Watch or reject Green
  if accounting/trust/channel/contract evidence is broken:
      route to 4C thesis-break rather than waiting for a full 4B overheat overlay
```

This is not a new global production change. It is a shadow-only R13 candidate for later batch ingestion.
