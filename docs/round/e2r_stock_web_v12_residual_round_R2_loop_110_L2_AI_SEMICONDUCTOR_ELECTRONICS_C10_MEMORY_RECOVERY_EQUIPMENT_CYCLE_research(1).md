## 5. Residual diagnosis

C10 is not a pure momentum bucket. The same memory-equipment recovery label produced two animals with the same skin: early cycle entries with a concrete equipment/customer/order route created large MFE asymmetry, while late-cycle relative-strength entries without order/revenue/margin bridge became severe 180D MAE traps. The hinge is not broad memory recovery itself; the hinge is whether that recovery has crossed from industry beta into company-level order, revenue, margin, or revision bridge.

The positive side is `031980`, `042700`, and `036810`: C10 can miss structural upside when it waits for fully visible earnings revision. The counterexample side is `036200`, `039440`, `079370`, and `101490`: late-cycle equipment RS without bridge produced shallow upside and deep forward drawdown. This argues for a C10-specific middle state, not a global Green threshold change.
