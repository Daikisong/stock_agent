# 1. 우리가 맞게 잡은 핵심

우리가 만든 E2R 에이전트의 핵심은 이거였지.

> **EPS 폭발 + 밸류에이션 리레이팅 + 아직 가격이 다 설명하지 못한 구간**

이건 문서의 핵심과 거의 같다. 문서도 서생원식 투자를 “앞으로 EPS 체급이 완전히 달라지는데, 시장은 아직 예전 산업으로 착각해서 싸게 보고 있는 구간”을 찾는 방식으로 정리하고 있다. 또 핵심 문장 자체도 “폭발적인 EPS 상승 후 밸류에이션 리레이팅되는 종목을 찾는다”로 요약돼 있다. 

우리가 변압기 백테스트에서 잡은 것도 이 구조였다.

| 우리가 본 것                                 | 문서의 원칙과 연결                |
| --------------------------------------- | ------------------------- |
| 효성중공업: 실적은 지저분하지만 저마진 수주 정리 후 EPS 폭발    | 현재 실적보다 앞으로의 EPS 체급 변화를 봄 |
| HD현대일렉트릭: 이미 +200% 올랐지만 수주·마진·밸류가 더 열림  | 이미 오른 종목 자동 탈락 금지         |
| 일진전기: CAPA 증설, 수주잔고, 고마진 프로젝트           | 공급 병목과 가격 결정력             |
| LS ELECTRIC: 전력기기에서 데이터센터 전력 인프라로 재분류   | 기존 업종에서 새 프레임으로 리레이팅      |
| 대한전선: 테마는 강하지만 PER/ROE 구조가 약해 E2R S급 탈락 | EPS 없이 멀티플만 오르는 테마주 배제    |

이건 정확히 문서의 “초과수익 = EPS 상향 × 밸류에이션 리레이팅 × 집중 보유” 공식과 맞다. 문서는 EPS만 오르면 평범한 사이클 투자이고, 멀티플만 오르면 테마주에 가깝다고 정리한다. 우리가 대한전선을 E2R 본류가 아니라 모멘텀형으로 분류한 이유도 이 원칙과 일치한다.

# 2. 우리가 특히 잘 맞게 잡은 부분

## 2-1. “안 오른 종목”이 아니라 “가격이 아직 다 설명하지 못한 종목”

이게 제일 중요하다.

우리는 백테스트에서 효성중공업이 발견 시점에 이미 저점 대비 +60%, HD현대일렉트릭이 +190~205%, 일진전기가 +235%, LS ELECTRIC이 +183%였어도 탈락시키지 않았다. 이건 아주 중요하다.

문서도 같은 말을 한다. 핵심 질문은 “지금 PBR이 과거 밴드 상단인가?”가 아니라, **“과거 PBR 밴드로 보는 것이 아직 맞는가?”**라고 정리한다. 즉 과거 밴드로 보면 비싸 보여도, 할인 요인이 제거되고 산업 성격이 바뀌면 아직 쌀 수 있다는 뜻이다. 

그래서 우리 에이전트의 철학은 이렇게 가야 한다.

```text
상승률이 높다 = 탈락
이 공식은 틀림.

상승률이 높은데도
EPS, 수주, 마진, FCF, 장기계약, 밸류 프레임 변화가
그 상승을 아직 다 설명하지 못한다
= 핵심 후보
```

이 부분은 방향이 정확하다.

---

## 2-2. 산업 수급을 종목보다 먼저 본 것

문서는 데이터에서 가장 많이 나온 관심사가 공급, 수요, 가격, CAPA, 쇼티지, 증설이었다고 정리한다. 실적, 밸류에이션, 장기계약, 리포트·컨콜도 핵심 정보원으로 잡혀 있다. 

우리가 설계한 파이프라인도 정확히 이 순서였다.

```text
산업 레짐 변화
→ 기업별 수주/CAPA/ASP/OPM 확인
→ EPS 컨센서스 상향
→ 밸류에이션 리레이팅
→ 가격 단계 판정
```

이건 맞는 방향이야.
에이전트가 “오늘 리포트에서 목표가 상향된 종목”만 찾으면 안 되고, **왜 목표가가 상향됐는지**, 그 뒤에 산업 수급 변화가 있는지까지 봐야 한다.

---

## 2-3. 컨센서스 상향을 후행이지만 강한 확인 신호로 본 것

우리가 Stage 2와 Stage 3를 나눴지.

```text
Stage 2: 수주, CAPA, EPS 가능성 보임
Stage 3: 실적 확인, 목표가 상향, 컨센서스 상향, 멀티플 상향
```

이것도 문서와 맞다. 문서는 컨센서스를 맹신하지는 않지만, 현실 변화가 업계 데이터, 컨콜·리포트, 컨센서스 상향을 거쳐 주가에 반영된다고 설명한다. 특히 중요한 것은 단기 실적 상향이 아니라 **2027년, 2028년 같은 중장기 숫자가 같이 올라가는지**라고 강조한다. 

따라서 우리가 만든 Stage 3는 맞다.
다만 부족한 게 있다. 현재 우리는 FY1/FY2 중심으로 봤는데, 이 문서의 정신대로라면 **FY3/FY4, 즉 3~4년 뒤 EPS/FCF 상향**까지 봐야 한다.

# 3. 부족한 부분 1: EPS뿐 아니라 FCF를 반드시 넣어야 한다

지금 우리 에이전트 이름은 E2R이었고, 주로 EPS/OP를 봤다.
하지만 문서는 EPS와 함께 **FCF/FCFF**를 반복적으로 강조한다. 핵심 결론에도 EPS/FCF가 폭발적으로 올라가는 구간이라고 정리되어 있다. 

그래서 점수식을 바꿔야 한다.

기존:

```text
EPS Explosion Score
= EPS 증가율 + OP 증가율 + OPM 개선 + ROE 개선
```

수정:

```text
Earnings & Cash Flow Explosion Score
= EPS 증가율
+ OP 증가율
+ FCF 증가율
+ FCF 전환율
+ OPM 개선
+ ROE 개선
+ 운전자본 악화 여부 감점
```

왜냐하면 EPS는 좋아 보이는데 현금이 안 들어오는 회사가 있다.
수주가 늘었는데 매출채권이 폭증하거나, CAPEX가 너무 커서 FCF가 안 나오거나, 재고가 쌓이면 겉보기 EPS와 실제 기업가치가 갈라진다.

특히 조선, 방산, 변압기, 반도체 장비, 플랜트는 반드시 봐야 한다.

| 항목     | 봐야 하는 이유              |
| ------ | --------------------- |
| 영업현금흐름 | 이익이 현금으로 들어오는지        |
| FCF    | 성장을 위해 돈을 너무 많이 태우는지  |
| 매출채권   | 수주/매출은 늘지만 회수가 안 되는지  |
| 재고     | 수요 둔화나 과잉생산 신호        |
| 선수금    | 고객이 미래 물량을 진짜로 묶고 있는지 |
| CAPEX  | 공급 확대가 너무 빨라지는지       |

즉, 앞으로 에이전트 점수는 **EPS Explosion**이 아니라 **EPS/FCF Explosion**으로 바꾸는 게 맞다.

# 4. 부족한 부분 2: “장기계약의 질”을 따로 점수화해야 한다

문서에서 굉장히 중요한 구분이 나온다.

> 일회성 쇼티지와 구조적 쇼티지는 다르다.
> 구조적 쇼티지는 앞으로 몇 년치 물량을 미리 묶어야 할 정도로 부족한 것이다. 

우리는 지금 수주잔고, CAPA, ASP를 봤지만, 아직 **계약의 질**을 충분히 분리하지 않았다.

같은 수주잔고라도 완전히 다르다.

| 구분       | 낮은 점수      | 높은 점수          |
| -------- | ---------- | -------------- |
| 계약 기간    | 단기         | 3~5년 이상        |
| 구속력      | 취소 가능      | 취소불가           |
| 선수금      | 없음         | 고객 선수금 있음      |
| 가격 구조    | 고정가, 원가 부담 | 가격 전가 가능       |
| 고객       | 불명확        | 빅테크, 정부, 대형 고객 |
| 매출 전환    | 불확실        | 납품 스케줄 명확      |
| 마진       | 낮음         | 고마진 프로젝트       |
| RPO/수주잔고 | 없음         | 증가 추세          |

그래서 새 에이전트에는 **Contract Quality Agent**가 필요하다.

```text
Contract Quality Score =
  장기계약 여부
+ 선수금 여부
+ 취소불가 여부
+ RPO/수주잔고 증가
+ 수주잔고/매출 비율
+ 고객 신용도
+ 가격 전가 조항
+ 고마진 여부
- 취소 가능성
- 단기성 수주
- 저마진 수주
```

이게 없으면 에이전트가 단순 “수주 공시 많은 종목”에 속을 수 있다.
서생원식으로는 “수주가 있다”가 아니라 **“몇 년짜리 이익 가시성이 잠겼다”**가 핵심이다.

# 5. 부족한 부분 3: “시장 오해”를 더 정밀하게 만들어야 한다

문서는 초과수익의 원천을 이렇게 본다.

```text
시장 다수의 기존 프레임이 틀림
→ 주가가 새 현실을 덜 반영함
→ 시간이 지나며 컨센서스와 밸류가 따라옴
→ 먼저 크게 산 사람이 초과수익을 가져감
```

그리고 단순히 시장 반대로 가는 게 아니라, 시장이 틀렸다고 판단하려면 더 잘 아는 산업 데이터가 있어야 한다고 강조한다. 

우리는 “리레이팅 Thesis Agent”를 만들었지만, 아직 **시장 오해를 계량화하는 방식**이 부족하다.

이걸 추가해야 한다.

## Market Mispricing / Old Frame Score

에이전트가 이렇게 질문해야 한다.

```text
1. 애널리스트들이 아직 과거 밸류 방식으로 보고 있는가?
2. 리포트 제목과 본문에 여전히 "시클리컬", "일회성", "피크아웃" 프레임이 강한가?
3. 실제로는 장기계약/수주잔고/선수금이 늘고 있는데, 목표 PER/PBR은 과거 밴드인가?
4. 스트리트 하이와 평균 추정치 차이가 큰가?
5. 소수 리포트만 새 프레임을 제시하고 있는가?
6. 기관 보유율이나 커버리지 수가 아직 낮은가?
7. 주가는 올랐지만 컨센서스가 아직 따라오지 못했는가?
```

이 점수가 높으면 좋은 의미다.
시장이 아직 완전히 이해하지 못했다는 뜻이니까.

반대로 이런 상태가 되면 기대수익률은 낮아진다.

```text
모든 리포트가 같은 논리로 긍정적
목표 PER/PBR이 이미 새 프레임을 충분히 반영
기관/외인 포지션이 과밀
중장기 컨센서스 상향이 둔화
주가만 계속 오름
```

문서도 모두가 새 프레임을 받아들이면 초과수익의 원천이 줄어든다고 정리한다.

# 6. 부족한 부분 4: Stage 4를 “졸업생” 하나로만 보면 안 된다

우리가 지금까지 Stage 4를 대체로 “졸업생/피크아웃 감시”로 불렀지.
이건 스캐너 입장에서는 맞지만, 서생원식 운용 입장에서는 약간 부족하다.

문서는 리레이팅 종목은 타이밍을 맞출 수 없으므로 미리 들고 있어야 하고, 리레이팅이 끝날 때까지 조정과 잡음을 버티는 전략이라고 정리한다. 단기 고점에서 20~30% 조정도 이상한 일이 아니라고 본다. 

그래서 Stage 4를 둘로 나눠야 한다.

| 단계                          | 의미                                           |
| --------------------------- | -------------------------------------------- |
| **Stage 4A: 리레이팅 진행/보유 구간** | 이미 많이 올랐지만 EPS·FCF·수주·장기계약이 계속 좋아짐. 보유 논리 유지 |
| **Stage 4B: 졸업/과열 구간**      | 주가와 멀티플이 너무 앞서고, 컨센서스 상향은 둔화                 |
| **Stage 4C: 피크아웃/논리 훼손**    | EPS 경로, 수주, 가격, 계약 구조가 깨짐                    |

이 구분이 중요하다.

예를 들어 HD현대일렉트릭이나 효성중공업이 크게 오른 뒤에도, 한동안은 Stage 4B가 아니라 **Stage 4A**였을 수 있다.
좋은 기업이 이미 올랐다고 바로 매도하는 게 아니라, **논리가 유지되는지**를 봐야 한다.

새 구조는 이렇게 가야 한다.

```text
Stage 1: 산업/기업 레이더
Stage 2: 후보 편입
Stage 3: 고확신 핵심 후보
Stage 4A: 리레이팅 진행, 보유 논리 유지
Stage 4B: 졸업생, 신규 진입 위험 증가
Stage 4C: 논리 훼손, 감축/제외
```

# 7. 부족한 부분 5: 매도/논리 훼손 에이전트가 더 중요하다

지금 우리는 “어떻게 찾을까”에 집중했어.
그런데 문서는 “언제 논리가 깨지는가”도 매우 중요하게 정리한다.

문서에서 유추된 매도 또는 논리 훼손 조건은 꽤 명확하다.

* 컨센서스 상향이 멈춘다.
* 스트리트 하이 추정치가 내려온다.
* 중장기 실적이 다시 플랫/역성장으로 바뀐다.
* LTA, 선수금, 계약 물량, RPO/수주잔고가 기대보다 약하다.
* 공급이 예상보다 빨리 늘어난다.
* AI CapEx나 미국 경제 같은 핵심 수요 기반이 깨진다. 

그러니까 에이전트는 매수 후보만 찾으면 안 된다.
이미 보유한 종목에 대해 매일 이렇게 알려줘야 한다.

```text
논리 유지: Green
- EPS/FCF 상향 지속
- 수주잔고 증가
- 장기계약 증가
- ASP 상승
- 공급 병목 유지

주의: Yellow
- 주가는 급등했지만 EPS 상향 둔화
- 신규수주 감소
- OPM 정체
- 리포트 톤은 긍정적이나 숫자 상향 없음

논리 훼손: Red
- 중장기 EPS 하향
- 수주잔고 감소
- 계약 취소/지연
- 공급 증설 앞당김
- 가격 하락
- 수요 기반 붕괴
```

이게 없으면 에이전트가 “좋은 종목 리스트”만 주고, 정작 언제 위험해지는지 못 알려준다.

---

# 8. 부족한 부분 6: 자금 성격과 변동성 감내를 시스템에 넣어야 한다

이건 투자 성과에 매우 중요하다.

문서는 최소 2년 이상 현금화가 필요 없는 돈으로 투자해야 한다고 강조한다. 좋은 종목이어도 20~30% 조정이 반복될 수 있고, 기간이 정해진 돈은 조정을 버티기 어렵기 때문이다. 

그러면 에이전트는 단순히 이렇게 말하면 안 된다.

```text
종목 A: S급
```

이렇게 말해야 한다.

```text
종목 A: S급
예상 보유 기간: 2~4년
예상 변동성: 고
중간 조정 허용폭: -25%~-35%
자금 성격: 2년 이상 필요 없는 자금에만 적합
강제 현금화 가능성이 있으면 핵심 비중 부적합
```

이 부분이 없으면 에이전트가 종목은 잘 찾아도, 사용자가 중간 조정에서 털릴 수 있다.
문서의 정신은 **“찾기”만이 아니라 “버티기”까지 포함**한다.

---

# 9. 부족한 부분 7: 서클 오브 컴피턴스 점수

문서에서 아주 중요한 대목은, 서생원이 변압기를 많이 공부했지만 직접 투자하지 않았다는 부분이다. 이유는 자신이 100% 확신하는 산업이 아니었기 때문이다. 문서는 집중투자의 전제로 산업을 깊게 이해하고, 수급·공급 병목, 경쟁사 한계, 컨센서스가 왜 틀렸는지, 틀렸을 때 무엇을 봐야 하는지 알아야 한다고 정리한다. 

그래서 에이전트에는 **Circle of Competence Score**가 필요하다.

이건 종목 점수가 아니라, 사용자와 종목의 적합도다.

```text
사용자가 산업 구조를 설명할 수 있는가?
공급 병목을 설명할 수 있는가?
경쟁사의 한계를 설명할 수 있는가?
컨센서스가 왜 틀렸는지 설명할 수 있는가?
논리가 깨지는 조건을 알고 있는가?
20~30% 조정 때 버틸 수 있는가?
```

예를 들어 같은 S급 후보라도:

| 사용자 이해도    | 최종 판단       |
| ---------- | ----------- |
| 산업을 깊게 이해  | 핵심 후보 가능    |
| 리포트만 읽은 수준 | 관찰/소액       |
| 테마명만 앎     | 제외 또는 공부 필요 |

이걸 넣어야 서생원식 집중투자의 정신과 맞는다.

# 10. 점수 체계를 문서 기준으로 다시 맞추면

문서에는 에이전트 점수화 예시가 있다.

```text
1. EPS 폭발성: 20점
2. 이익 지속성/가시성: 20점
3. 공급 병목/가격 결정력: 20점
4. 시장 오해/컨센서스 후행: 15점
5. 밸류에이션 리레이팅 여지: 15점
6. 자본배분/주주환원: 5점
7. 정보 신뢰도/검증 가능성: 5점
```

감점은 수요 붕괴, 공급 과잉, 계약 구속력 부족, 밸류에이션 과열, 자금 기간 불일치다. 

나는 이걸 거의 그대로 채택하는 게 좋다고 봐.
우리가 만든 점수식보다 이쪽이 서생원식 정신에 더 가깝다.

## E2R Agent 2.0 점수

| 항목                | 점수 | 세부 내용                              |
| ----------------- | -: | ---------------------------------- |
| **EPS/FCF 폭발성**   | 20 | FY1~FY4 EPS, OP, FCF 증가율           |
| **이익 지속성/가시성**    | 20 | LTA, 선수금, RPO, 수주잔고, 계약 기간         |
| **공급 병목/가격 결정력**  | 20 | CAPA 제약, 리드타임, ASP, 고객 구매 압박       |
| **시장 오해/컨센서스 후행** | 15 | 낡은 프레임, 스트리트 하이/로우, 커버리지 변화        |
| **밸류에이션 리레이팅 여지** | 15 | fPER, fPBR, ROE/PBR, 과거 밴드 무효화 가능성 |
| **자본배분/주주환원**     |  5 | 자사주, 배당, 증설 효율, 희석 리스크             |
| **정보 신뢰도/검증 가능성** |  5 | 공시, 컨콜, 리포트, 산업 데이터의 일치도           |

그리고 감점:

| 감점 항목     | 내용                            |
| --------- | ----------------------------- |
| 수요 붕괴     | AI CapEx 붕괴, 경기 급락, 고객사 투자 축소 |
| 공급 과잉     | 신규 FAB, 경쟁사 CAPA, 중국 추격       |
| 계약 구속력 부족 | LTA가 실제 강제력이 약함               |
| 밸류 과열     | EPS보다 PER/PBR이 먼저 과도하게 상승     |
| 자금 기간 불일치 | 사용자가 버틸 수 없는 변동성              |
| 현금흐름 악화   | EPS는 좋은데 FCF 부진               |
| 희석 리스크    | CB/BW, 유상증자 반복                |
| 피크아웃      | 수주잔고, ASP, OPM, 컨센서스 둔화       |

이 점수 체계가 문서와 가장 잘 맞다.

# 11. 우리가 만든 Stage와 문서의 정신을 합치면

최종 Stage는 이렇게 가야 한다.

## Stage 0: 산업 가설

아직 종목이 아니라 산업을 본다.

```text
수요가 구조적으로 커지는가?
공급이 빨리 못 늘어나는가?
고객이 장기계약/선수금까지 줄 정도인가?
가격 결정력이 생기는가?
```

## Stage 1: 기업 레이더

기업 이벤트가 생긴다.

```text
신규시설투자
수주 공시
수주잔고 증가
ASP 상승 언급
고부가 제품 비중 증가
커버리지 개시
```

## Stage 2: 후보 편입

숫자와 스토리가 같이 보인다.

```text
EPS/OP/FCF 상향 가능성
수주/CAPA/마진 개선
밸류 아직 감당 가능
시장 오해 존재
```

## Stage 3: 고확신 후보

리레이팅이 실제로 시작된다.

```text
실적 서프라이즈
중장기 컨센서스 상향
목표주가 상향
적용 PER/PBR 상향
장기계약/RPO/수주잔고 확인
```

## Stage 4A: 리레이팅 진행, 보유 논리 유지

이미 많이 올랐지만 아직 논리가 살아 있다.

```text
EPS 계속 상향
FCF 좋아짐
수주잔고 증가
공급 병목 유지
밸류 프레임 변화 지속
```

## Stage 4B: 졸업생

좋지만 신규 기대수익률은 낮아진다.

```text
모두가 새 프레임을 인정
밸류가 충분히 높아짐
EPS 상향 속도 둔화
추가 오해 해소 여지 작음
```

## Stage 4C: 논리 훼손

나와야 하는 구간이다.

```text
중장기 EPS 하향
계약 구속력 약화
수주잔고 감소
ASP 하락
공급 과잉
수요 붕괴
```

이렇게 해야 문서의 “리레이팅이 끝날 때까지 보유”와 “논리가 깨질 때까지 버팀”이 동시에 구현된다.

# 12. 최종 판단: 방향은 맞고, 보강해야 할 건 명확하다

## 맞는 방향

우리가 설계한 에이전트는 이미 문서의 핵심과 잘 맞는다.

```text
구조 변화
→ EPS/FCF 체급 변화
→ 시장의 과거 프레임
→ 밸류에이션 리레이팅
→ 가격이 이미 올라도 자동 탈락 금지
→ 논리가 유지되면 보유
```

문서도 최종적으로 서생원식 매매를 **구조 변화, EPS 폭발, 시장 오해, 집중 보유, 리레이팅**으로 정리한다. 

## 부족한 부분

하지만 지금 버전은 아직 아래가 부족하다.

| 부족한 부분      | 보강 방향                              |
| ----------- | ---------------------------------- |
| EPS 중심      | FCF/FCFF까지 포함                      |
| 수주잔고 단순 확인  | 계약의 질, 선수금, 취소불가, RPO 확인           |
| FY1/FY2 중심  | FY3/FY4 중장기 컨센서스까지 확인              |
| 리포트 기반 판단   | 컨콜, 산업 가격, 고객 CapEx, 경쟁사 CAPA까지 확장 |
| 발견 중심       | 보유 논리 점검, 논리 훼손 감지 추가              |
| Stage 4 단순화 | Stage 4A 보유, 4B 졸업, 4C 훼손으로 분리     |
| 종목 점수 중심    | 사용자 자금 기간·서클 오브 컴피턴스 반영            |
| 테마 리스크      | Red Team Agent 강화                  |

# 13. 결론

우리가 가는 방향은 맞다.
다만 에이전트의 이름을 더 정확히 붙이면 **“종목 추천 에이전트”가 아니라 “산업 리레이팅 탐지 + 논리 유지/훼손 감시 에이전트”**여야 한다.

최종 설계 철학은 이거다.

```text
오늘 오를 종목을 찾지 않는다.

몇 년짜리 이익 체급 변화가 생겼는데,
시장은 아직 과거 프레임으로 보고 있고,
EPS/FCF와 장기계약/수주/공급 병목이 그 변화를 증명하며,
밸류에이션 기준 자체가 바뀔 종목을 찾는다.

그리고 찾은 뒤에는
주가가 아니라 논리가 깨지는지를 매일 감시한다.
```

이렇게 만들면 지금 문서에 담긴 정신과 거의 정확히 맞는다.
