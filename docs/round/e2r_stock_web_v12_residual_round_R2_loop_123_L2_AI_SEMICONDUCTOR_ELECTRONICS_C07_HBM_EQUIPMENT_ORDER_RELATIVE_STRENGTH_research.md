## 17. Sector-Specific Rule Candidate

**L2_HBM_EQUIPMENT_RELATIVE_STRENGTH_ORDER_CONVERSION_GATE**: Within L2 semicap equipment, relative strength plus HBM vocabulary is enough for Stage2-watch, and relative strength plus customer evaluation/product route can be Stage2-Actionable. A second large order after a price run-up is evaluated as 4B overlay first.

## 18. Canonical-Archetype Rule Candidate

**C07_ORDER_CONVERSION_BEFORE_GREEN_RELATIVE_STRENGTH_STAGE2_ONLY_GATE**: C07 is not a pure momentum bucket. It is a three-tier ladder: Tier A confirmed order/supply conversion; Tier B qualified product/customer testing route; Tier C proxy expectation. Tier A can support Actionable/Yellow, Tier B can support Actionable but blocks Green, Tier C is watch-only and becomes a counterexample if promoted.
