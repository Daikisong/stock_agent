## 4. Narrative findings

### 4.1 Direct conversion wins: Daeduck and Wonik Materials

Daeduck Electronics is useful because it was not a clean HBM winner in early 2024. The November/December 2024 evidence is subtler: memory demand had concentrated in HBM and ordinary DRAM was weak, yet DDR5 mix still gave Daeduck a measurable memory-revenue recovery direction. The price path from `2024-12-02` confirms that a bottoming-and-reopen trigger can work when the evidence admits the weakness and identifies a recovery bridge rather than hiding behind generic memory beta.

Wonik Materials is the cleanest consumables/materials extension in this loop. It is not equipment PO, but the special-gas route has direct process-line linkage: semiconductor memory recovery, customer line expansion, inventory/cost improvement, and AI semiconductor demand. Its path has modest early MAE but strong 180D MFE, so C10 should not be narrowed to equipment orders only. The rule should allow consumables/materials when utilization and margin bridge are explicit.

### 4.2 Good story, bad full window: TLB and Simmtech

TLB had an understandable Stage2-Actionable entry: server DDR5 module PCB, customer DDR5 supply growth, and data-center/server demand. The first 90 trading days delivered MFE above 36%, but the 180D MAE fell below -50%. This is exactly the kind of case where C10 should attach local 4B after early MFE instead of letting a DDR5 label become durable Green.

Simmtech shows the weaker version of the same pattern. A recovery note can be right directionally, but if the company is still an ordinary memory substrate/PCB proxy and is excluded from the high-margin HBM substrate bridge, the full-window drawdown can erase the thesis. The row is not a rejection of memory substrate recovery; it is a guard against treating all substrate recovery as equal.

### 4.3 Boundary route needs discount: SFA Semicon and Dongjin Semichem

SFA Semicon's OSAT recovery language had customer names and BEP expectation, but the path was almost no upside and a deep MAE. It should be Stage2-watch unless the OSAT volume and margin conversion are booked and visible. Dongjin Semichem had a real process-material route and produced meaningful 90D upside, but it belongs in C10 only with boundary discount because EUV PR/CMP slurry/hybrid bonding materials are process-enabling signals rather than direct memory equipment order-cycle evidence.

## 5. Proposed rule candidate

```text
rule_scope: canonical_archetype_specific
canonical_archetype_rule_candidate: C10_MEMORY_RECOVERY_REQUIRES_DIRECT_UTILIZATION_ORDER_REVENUE_MARGIN_BRIDGE_WITH_BOUNDARY_ROUTE_AND_FULL_WINDOW_4B_CAP
```

Mechanism:

1. Give C10 Stage2-watch for broad memory recovery beta.
2. Upgrade to Stage2-Actionable only when one of the following is explicit: customer capacity recovery, named order/PO, utilization recovery, revenue recognition, margin bridge, or consumables demand tied to customer line expansion.
3. Apply a boundary discount to substrate, OSAT, process-material, and special-gas routes unless they show direct memory customer conversion.
4. If 30/90D MFE is large but 180D MAE opens deeply, treat it as local 4B/profit-lock, not durable Yellow/Green.
5. Do not turn ordinary-memory recovery into hard 4C unless repeated non-price thesis break appears. This loop is mostly 4B/false-positive, not hard 4C.
