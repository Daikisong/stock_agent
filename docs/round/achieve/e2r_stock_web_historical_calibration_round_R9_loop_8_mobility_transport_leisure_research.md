### 9.3 대한항공 — cargo margin bridge after corporate-action window

- HMM: Stage2/Stage2-Actionable created the best entry. The mechanism was not “theme only”; it was freight-rate evidence plus OP leverage plus relative strength.

- 제주항공: traffic recovery evidence was real, but price failed to rerate. This is an execution/fuel/dilution-overhang guardrail.

## 24. Score-Price Alignment Verdict

This round supports a more surgical early-entry rule: **Stage2-Actionable should activate when real rate/margin/revision evidence is paired with relative strength**. It should not activate on reopening policy narrative alone.

The working metaphor: shipping cycle evidence behaved like a loaded container ship already leaving port; by the time Stage3-Green stamped the manifest, part of the voyage had already happened. Reopening-only travel names behaved like passengers gathered at the gate while the aircraft still lacked fuel, crew, and clearance.

- Stage3-Green lateness detection for transport-cycle names.

- Split 4B local price-only watch vs. full-window non-price 4B.

### this_round_does_not_validate

- Hard 4C success.

- Airline full-cycle 4B exit using only price.
