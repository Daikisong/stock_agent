삼전 하닉은 상승릴레이, 사이클 뜨기전에 어땠어? 

지금 기준으로? 폭발적인 EPS 상승 후 리레이팅에 밸류까지 좋았었나?

맞아. **삼전·하닉은 사이클 뜨기 전에는 지금처럼 ‘화려한 성장주’처럼 보이지 않았다.**
오히려 겉으로는 실적 망가진 경기민감주처럼 보였고, 그래서 제대로 봐야 했던 지표가 **후행 PER이 아니라 PBR, 정상화 EPS, 업황 반전 신호**였어.

핵심부터 말하면:

> **삼전·하닉은 상승 초입에 “후행 실적 기준으로는 별로 안 싸 보이거나, 심지어 적자라 PER이 의미 없었지만”,
> 미래 EPS가 폭발할 걸 본 사람에게는 밸류가 매우 좋았다.
> 특히 삼성전자는 2024년 PBR 0.92배 구간, SK하이닉스는 2024년 PER 6배대·PBR 1.6배대 구간이 진짜 좋은 자리였다.**

# 1. 사이클 뜨기 전의 모습: 겉으로는 망가져 있었다

메모리 사이클 바닥에서는 항상 똑같은 현상이 나옵니다.

실적은 최악입니다.
재고는 많습니다.
DRAM·NAND 가격은 떨어집니다.
증권사는 목표가를 낮춥니다.
개미들은 “반도체 끝났다”고 합니다.

그런데 주가는 바닥에서 먼저 움직입니다. 왜냐하면 주식시장은 **현재 EPS가 아니라 다음 EPS**를 사기 때문입니다.

2023년을 보면 삼성전자는 연간 매출 258.9조 원, 영업이익 6.57조 원까지 내려왔고, DS 반도체 부문은 4분기에도 2.18조 원 영업손실을 냈습니다. 다만 그때 이미 삼성은 DRAM 흑자 전환, HBM·DDR5·서버 SSD 대응, 재고 소진 가속을 말하기 시작했습니다. 이게 바닥 신호였습니다. ([Samsung Global Newsroom][1])

SK하이닉스는 더 심했습니다. 2023년 연간 매출 32.77조 원, 영업손실 7.73조 원, 순손실 9.14조 원이었습니다. 그런데 2023년 4분기에 1년 만에 분기 영업흑자로 돌아섰고, 회사는 DDR5와 HBM3 매출이 전년 대비 각각 4배 이상, 5배 이상 증가했다고 발표했습니다. 이게 하닉의 리레이팅 서막이었습니다. ([PR Newswire][2])

즉, 초입의 모양은 이랬습니다.

| 구분     | 겉으로 보이는 모습       | 진짜 봐야 했던 것                        |
| ------ | ---------------- | --------------------------------- |
| 삼성전자   | 반도체 이익 급감, DS 적자 | DRAM 가격 반등, HBM·DDR5·서버 SSD 믹스 개선 |
| SK하이닉스 | 연간 적자, PER 계산 불가 | HBM3, DDR5, AI 서버 메모리 매출 폭증       |
| 시장 분위기 | “반도체 끝난 거 아닌가?”  | “재고 줄고 가격 오르면 EPS가 폭발한다”          |

# 2. 후행 PER로 보면 안 됐다

이게 제일 중요합니다.

메모리 바닥에서 후행 PER을 보면 착시가 생깁니다.
왜냐하면 EPS가 바닥이거나 적자니까 PER이 높게 보이거나 아예 의미가 없어집니다.

삼성전자 2023년 EPS는 2,131원까지 내려왔고, 2023년 PER은 36.84배로 높아 보였습니다. 그런데 2024년 EPS는 4,950원, 2025년 EPS는 6,564원, 2026E EPS는 41,972원으로 컨센서스가 급격히 올라가는 구조였습니다. 즉 2023년 후행 PER 36배만 보면 “비싸다”였지만, 2026년 EPS를 보면 완전히 다른 그림이었습니다. ([네이버 금융][3])

SK하이닉스는 더 극단적입니다. 2023년 EPS가 -12,517원이어서 PER 자체가 의미 없었습니다. 그런데 2024년 EPS 27,182원, 2025년 EPS 58,955원, 2026E EPS 290,404원으로 바뀌었습니다. 2024년 기준 PER은 6.40배였고, 2026E PER도 6배대입니다. 이게 바로 “EPS 폭발을 먼저 본 사람”에게 보였던 기회였습니다. ([네이버 금융][4])

그러니까 정답은 이거야.

> **삼전·하닉은 사이클 초입에 후행 PER로는 예뻐 보이지 않았다.
> 하지만 forward EPS, PBR, 업황 반전 신호로 보면 밸류가 좋았다.**

# 3. 그럼 밸류까지 좋았냐?

## 삼성전자는 확실히 좋았다

삼성전자는 특히 2024년이 매우 좋은 자리였습니다.

Naver/FnGuide 기준으로 삼성전자는 2024년 EPS 4,950원, PER 10.75배, PBR 0.92배였습니다. 메모리 업황은 바닥을 지나고 있었고, AI 메모리 수요는 살아나고 있었는데, 시장은 여전히 삼성전자를 “HBM에서 뒤처진 대형 반도체주”로 싸게 보고 있었습니다. ([네이버 금융][3])

이게 진짜 좋은 구조입니다.

| 구분        | 삼성전자 2024년 |
| --------- | ---------: |
| EPS       |     4,950원 |
| PER       |     10.75배 |
| PBR       |      0.92배 |
| ROE       |      9.03% |
| 2026E EPS |    41,972원 |
| 2026E ROE |     50.43% |

PBR 0.92배에서 ROE가 50%대로 올라갈 그림이 보이면, 이건 그냥 싸다 수준이 아닙니다.
시장 입장에서는 “자본 1원당 이익을 별로 못 내는 회사”로 가격 매기던 종목이 갑자기 “자본을 미친 듯이 회전시키는 AI 메모리 병목 기업”으로 보이기 시작하는 겁니다.

그래서 삼성전자는 **밸류까지 좋았다**고 보는 게 맞습니다.

---

## SK하이닉스도 좋았지만, 더 ‘사이클 선행형’이었다

SK하이닉스는 삼성전자보다 순도가 높았습니다. HBM 본체였기 때문입니다.

2023년에는 적자라서 PER로 판단할 수 없었습니다. 하지만 2024년에는 EPS 27,182원, PER 6.40배, PBR 1.62배, ROE 31.06%로 바뀌었습니다. 2026E EPS는 290,404원, ROE는 93.62%까지 잡혀 있습니다. ([네이버 금융][4])

| 구분        | SK하이닉스 2024년 |
| --------- | -----------: |
| EPS       |      27,182원 |
| PER       |        6.40배 |
| PBR       |        1.62배 |
| ROE       |       31.06% |
| 2026E EPS |     290,404원 |
| 2026E ROE |       93.62% |

이건 엄청난 구조입니다.
ROE 30%짜리 회사가 PER 6배, PBR 1.6배에 있었다는 건, 지금 와서 보면 거의 교과서적인 리레이팅 초입입니다.

다만 SK하이닉스는 삼성전자보다 더 빨리 시장이 알아봤습니다. HBM3, HBM3E, 엔비디아 밸류체인, AI 서버 메모리라는 키워드가 너무 직접적이었기 때문입니다.

# 4. 지금은 어떤 상태냐?

지금은 이미 상당히 리레이팅이 진행됐습니다.

삼성전자는 2026년 1분기에 매출 133.9조 원, 영업이익 57.2조 원을 냈고, DS 반도체 부문만 영업이익 53.7조 원을 기록했습니다. 메모리 가격 상승과 AI 고부가 수요가 실적을 밀었습니다. ([Samsung Global Newsroom][5])

SK하이닉스도 2026년 1분기 매출 52.576조 원, 영업이익 37.610조 원, 순이익 40.346조 원으로 사상 최대 실적을 냈습니다. ([SK hynix Newsroom][6])

현재 Naver 기준으로 삼성전자는 현재가 279,000원, 52주 저가 53,700원, 52주 고가 291,500원입니다. SK하이닉스는 현재가 1,835,000원, 52주 저가 196,500원, 52주 고가 1,967,000원입니다. 즉 이미 주가가 크게 움직인 상태입니다. ([네이버 금융][3])

그래서 지금은 이렇게 봐야 합니다.

| 구분     | 과거 초입                       | 지금                                 |
| ------ | --------------------------- | ---------------------------------- |
| 삼성전자   | PBR 1배 이하, HBM 의심, EPS 회복 전 | 실적 폭발 확인, forward PER 낮지만 PBR은 높아짐 |
| SK하이닉스 | 적자에서 흑자 전환, HBM 선도 확인 중     | HBM 대장으로 인정, EPS 폭발 확인, 주가도 크게 상승  |
| 밸류     | 싸고 의심 많음                    | PER은 아직 싸 보이나, PBR·기대치는 많이 올라감     |
| 리스크    | “진짜 회복하나?”                  | “너무 많이 반영한 건 아닌가?”                 |

# 5. 현재 밸류는 아직 좋은가?

여기서 구분해야 합니다.

## Forward PER만 보면 아직 좋아 보인다

현재 삼성전자의 추정 PER은 약 7배, 추정 EPS는 41,972원으로 제시됩니다. SK하이닉스는 추정 PER 약 6배, 추정 EPS 290,404원입니다. ([네이버 금융][3])

이 숫자만 보면 “아직 싸네?”라고 볼 수 있습니다.

## 그런데 PBR로 보면 이미 리레이팅이 많이 됐다

현재 Naver의 최근 BPS 기준 PBR은 삼성전자 4.36배, SK하이닉스 10.51배로 표시됩니다. 다만 2026E BPS까지 반영하면 삼성전자는 약 2.7배, SK하이닉스는 약 4배 수준으로 낮아집니다. 그래도 2024년 삼성전자 PBR 0.92배, SK하이닉스 PBR 1.62배와 비교하면 이미 리레이팅이 많이 진행된 상태입니다. ([네이버 금융][3])

즉 현재는:

> **PER은 싸 보인다.
> 하지만 PBR과 기대치는 이미 많이 올라왔다.
> 그래서 과거 초입처럼 “밸류까지 압도적으로 싼 구간”은 아니다.**

# 6. 삼전·하닉의 진짜 매수 포인트는 언제였나?

정확히 말하면, 가장 좋은 구간은 **실적은 아직 더러운데 업황 반전 신호가 나오던 시기**였습니다.

예를 들면 이런 구간입니다.

| 신호                    | 의미                 |
| --------------------- | ------------------ |
| DRAM/NAND 가격 하락폭 둔화   | 바닥 근접              |
| 감산, 설비투자 축소           | 공급 조절 시작           |
| 재고 감소                 | 가격 반등 가능성          |
| HBM·DDR5·서버 SSD 매출 증가 | 제품 믹스 개선           |
| 분기 적자 축소              | EPS 턴어라운드 초입       |
| 컨센서스 상향 시작            | 주가가 먼저 움직일 준비      |
| PBR 1배 안팎             | 시장이 아직 회복을 안 믿는 구간 |

이때가 가장 좋습니다.

하닉은 2023년 4분기에 1년 만에 영업흑자로 돌아섰고, HBM3와 DDR5 매출 폭증을 발표했습니다. 삼성은 2023년 4분기에 DRAM 흑자 전환, HBM·DDR5·서버 SSD 대응을 말했습니다. 이때부터 “후행 실적은 최악인데 다음 실적은 달라진다”는 그림이 나온 겁니다. ([PR Newswire][2])

# 7. 그래서 네 질문에 대한 정확한 답

## “삼전 하닉은 사이클 뜨기 전에 어땠어?”

**실적은 망가져 있었고, 분위기는 안 좋았다.**
삼성은 반도체 부문 적자, 하닉은 연간 적자였다.
하지만 가격 반등, 재고 소진, HBM·DDR5 수요, AI 서버 수요가 먼저 보이기 시작했다.

## “폭발적인 EPS 상승 후 리레이팅에 밸류까지 좋았었나?”

**맞다. 특히 2024년 기준으로는 밸류까지 좋았다.**

삼성전자는 PBR 0.92배에서 2026E ROE 50%대로 가는 그림이었다.
SK하이닉스는 PER 6배대, PBR 1.6배대에서 2026E EPS가 10배 이상 커지는 그림이었다.
이건 “EPS 폭발 + 밸류 저평가 + 업종 재분류”의 정석이었다.

## “지금도 그런가?”

**EPS 폭발은 지금도 맞다.
하지만 밸류까지 초입처럼 좋냐고 하면, 그건 아니다.**

지금은 실적이 이미 확인됐고, 주가도 이미 크게 올라왔다.
그래서 현재의 삼전·하닉은 **초입 발굴주**라기보다 **메모리 슈퍼사이클의 대장주**입니다.

# 8. 이걸 지금 종목 발굴에 적용하면

삼전·하닉에서 배워야 할 건 이겁니다.

> **좋은 자리는 실적이 이미 예쁜 때가 아니라,
> 실적은 아직 더럽지만 다음 EPS가 폭발할 증거가 쌓이는 때다.**

그래서 지금 새로운 후보를 찾을 때는 이런 종목을 봐야 합니다.

| 조건                         | 설명             |
| -------------------------- | -------------- |
| 현재 실적은 애매하거나 막 턴어라운드       | 아직 시장이 확신하지 못함 |
| 2026~2027 EPS가 3배 이상 증가 가능 | 단순 회복이 아니라 레벨업 |
| PBR이 아직 과하게 높지 않음          | 리레이팅 여지가 남음    |
| 수주·가격·믹스 개선이 확인됨           | EPS 추정치 상향의 근거 |
| 시장이 아직 기존 업종으로 보고 있음       | 재분류될 여지가 있음    |

삼전·하닉은 이제 “그런 종목을 찾는 기준점”입니다.
현재 삼전·하닉 자체는 여전히 좋지만, **가장 큰 수익률 구간은 이미 ‘의심받던 초입’에서 나왔다**고 보는 게 맞습니다.

한 문장으로 압축하면:

> **삼전·하닉은 사이클 전에는 후행 실적으로는 별로였지만, forward EPS와 PBR로 보면 아주 좋았다.
> 지금은 EPS 폭발은 맞지만, 밸류 리레이팅은 이미 상당히 진행된 대장주 상태다.**
