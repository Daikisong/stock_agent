지금 너가 말해준대로 했을때 이렇게 인식되는 에이전트가 만들어지는거야? 말해준대로 한번더 백테스트해줘

응. **지금 말한 공정대로 만들면 스샷처럼 “이미 올랐는데도 대어를 잡는 에이전트”가 만들어지는 게 맞아.**
다만 스샷 표에는 꼭 한 가지를 추가해야 해.

> **인식 시점 = 하나가 아니라 두 개다.**
> **Stage 2 후보 편입 시점**과 **Stage 3 고확신 시점**을 따로 찍어야 한다.

왜냐하면 효성중공업처럼 2023년 5월 2일에 이미 잡히긴 했지만, 그건 “후보 편입”이고, 2023년 7월 10일~7월 31일쯤에는 “실적·마진·목표가 상향이 확인된 고확신”으로 한 단계 올라갔기 때문이야.

# 1. 이번 재백테스트 기준

이번에는 내가 이전에 말한 공정 그대로 다시 봤어.

기준은 이거야.

1. **그 날짜에 이미 공개된 정보만 사용**
2. **리포트의 현재주가와 52주 고저를 당시 가격 데이터로 사용**
3. **당시 이미 오른 정도 = 현재주가 ÷ 리포트상 52주 저가 - 1**
4. **그 뒤 결과 = 2026년 5월 13일 전후 현재가와 52주/연중 고점 기준**
5. **LS ELECTRIC은 2026년 4월 5:1 액면분할을 반영해 가격을 조정**
6. **Stage 1~4를 분리해서 판단**

실제 자동화에서는 KRX Open API로 주가·거래대금·시총을 가져오고, OpenDART로 공시·재무제표를 가져오고, FnGuide/QuantiWise로 EPS 컨센서스와 상향률을 가져와야 해. KRX Open API는 유가증권 일별매매정보를 제공하고 2010년 이후 데이터를 대상으로 하며, OpenDART는 단일회사 전체 재무제표 API와 보고서 코드 체계를 제공한다. FnGuide QuantiWise는 90일 이내 리포트 기반 컨센서스와 익일 오전 7시 30분경 업데이트되는 일간 컨센서스를 제공한다. 뉴스는 네이버 뉴스 검색 API처럼 JSON/XML로 검색 결과를 받을 수 있는 API를 붙이면 된다. ([Open API][1])

# 2. Stage 정의를 다시 정확히 잡으면

| 단계                     | 의미                                | 에이전트 판단  |
| ---------------------- | --------------------------------- | -------- |
| **Stage 1 레이더**        | 산업 레짐 또는 기업 이벤트만 발생               | 워치리스트    |

| **Stage 4 졸업/피크아웃 감시** | 너무 많이 반영됐거나 이익 모멘텀 둔화             | 신규 발굴 아님 |

스샷에 나온 “에이전트가 인식했을 시점”은 대부분 **Stage 2 기준**이야.
실전 브리핑에서는 반드시 이렇게 나와야 해.

```text
효성중공업
Stage 2 후보 편입: 2023.05.02
Stage 3 고확신: 2023.07.10~07.31
```

이렇게 분리해야 실제 매매 판단이 훨씬 정확해져.

# 5. 종목별로 실제 에이전트가 어떻게 판단했는지

## 5-1. 효성중공업

### Stage 2 후보 편입: 2023년 5월 2일

검색어는 이거였어.

```text
효성중공업 저마진 수주 정리 1Q 거의 완료 갈수록 좋다 2023 5월 2일 pdf
```

찾은 리포트의 핵심은 이거야.

* 제목: **저마진 수주 정리 1Q 거의 완료. 갈수록 좋다**
* 현재주가: 78,500원
* 52주 고/저: 89,800원 / 49,050원
* 2024E EPS: 13,368원
* 2025E EPS: 18,739원
* 2024E PER: 5.9배
* 2025E PER: 4.2배
* 2024E PBR: 0.5배
* 2025E PBR: 0.4배

리포트는 1Q23 부진이 실질적 부진이라기보다 2020년 이전 저마진·역마진 수주의 정리 과정이라고 설명했고, 중공업 부문 영업이익률이 2Q23부터 회복될 것으로 봤다. 당시 forward PER/PBR도 매우 낮았다. ([Naver][2])

에이전트 판정은 이렇게 나와야 해.

```text
효성중공업
Stage: 2
등급: A+
인식일: 2023.05.02
인식 이유:
- 현재 실적은 지저분하지만 원인이 저마진 수주 정리
- 2024E/2025E EPS 급증
- 2024E PER 5.9배, 2025E PER 4.2배
- PBR 0.5배 이하
- 전력기기 호황 사이클 초입
```

이건 정확히 “하닉 2023년 10월형”이야.
실적표는 아직 깨끗하지 않은데, 다음 EPS가 보이는 상태.

### Stage 3 고확신: 2023년 7월 10일~7월 31일

7월 10일 리포트는 제목부터 **“호황 초입, 실적 호조. 밸류 프리미엄 시기”**였고, 2Q23 영업이익이 컨센서스를 대폭 초과할 것으로 전망했다. 목표주가도 100,000원에서 155,000원으로 55% 상향됐고, 현재주가는 121,900원이었다. ([Naver][3])

7월 31일 리포트는 더 강해. 목표주가를 190,000원으로 기존 대비 97.9% 상향했고, 2Q23 실적이 컨센서스를 상회했으며, 중공업 부문이 흑자전환했고, 고마진 수주잔고가 매출로 인식되며 마진이 8.0%로 개선됐다고 봤다. ([하나증권][4])

에이전트는 여기서 이렇게 상향해야 해.

```text
효성중공업
Stage 2 → Stage 3 상향
인식일: 2023.07.10 또는 2023.07.31
상향 이유:
- 컨센서스 초과 실적
- 저마진 수주 정리 완료
- 중공업 마진 회복
- 목표주가 55%~97.9% 상향
- 기존 중공업/건설 복합기업 할인 해소 시작
```

결론: **효성중공업은 에이전트가 가장 잘 잡았을 종목.**
5월 2일엔 후보 편입, 7월엔 고확신으로 올렸어야 해.

## 5-2. HD현대일렉트릭

### Stage 2 후보 편입: 2023년 7월 5일

검색어는 이거였어.

```text
HD현대일렉트릭 단기 변동성 우려 갈 길이 아직 먼데 무엇이 두려우랴 2023 7월 5일 pdf
```

리포트 핵심은 이거야.

* 현재주가: 67,300원
* 52주 고/저: 67,300원 / 22,050원
* 2Q23E 매출: 6,930억원
* 2Q23E 영업이익: 577억원
* 2Q23E 영업이익 YoY: +112%
* 목표주가: 67,000원 → 89,000원, +33% 상향

리포트는 2023년 이후 수주가 대부분 lead time 2~3년 이상인 장기 물량 중심이고, 장기 물량 수주 증가가 2024년 이후 실적 전망치 상향 요인이라고 설명했다. 목표주가도 실적 전망치와 멀티플 상향으로 33% 올렸다. ([Naver][5])

이때 이미 52주 저점 대비 +205%였어.
그런데도 에이전트는 탈락시키면 안 됐어. 이유는 주가가 아니라 **EPS·수주·멀티플이 더 크게 열리고 있었기 때문**이야.

```text
HD현대일렉트릭
Stage: 2
등급: A+
인식일: 2023.07.05
인식 이유:
- 이미 52주 신고가지만 실적 전망치 상향
- 장기 수주 물량 증가
- 목표주가 33% 상향
- 전력기기 산업 초호황
- 리포트가 직접 “밸류에이션도 아직 갈 길이 멀다”고 판단
```

### Stage 3 고확신: 2023년 7월 27일

검색어는 이거.

```text
HD현대일렉트릭 줄을 서시오 2023 7월 27일 pdf
```

이 리포트는 Stage 3 신호가 아주 명확했어.

* 현재주가: 69,600원
* 목표주가: 87,000원
* 상승여력: 25%
* 52주 고/저: 80,600원 / 24,000원
* 2Q23 영업이익 YoY: +116%
* 신규수주: 2Q23 12.3억달러, +38%
* 수주잔고: 37.2억달러, +48%
* 목표주가 21% 상향

리포트는 2Q23에 이익률이 눈에 띄게 증가했고, 수주가 계속 쌓이고 있으며, 수주잔고가 37.2억달러로 늘었다고 설명했다. 

에이전트 출력은 이렇게 나와야 해.

```text
HD현대일렉트릭
Stage 2 → Stage 3 상향
인식일: 2023.07.27
등급: S-
상향 이유:
- 2Q23 실적 확인
- 영업이익률 개선
- 수주잔고 37.2억달러
- 목표주가 21% 상향
- 전력기기주 → 글로벌 전력망 병목주로 재분류
```

결론: **스샷의 HD현대일렉트릭 인식은 맞다.**
다만 더 정확히는 7월 5일 Stage 2, 7월 27일 Stage 3.

## 5-3. 일진전기

### Stage 1 레이더: 2023년 9월 8일

검색어는 이거였어.

```text
일진전기 신규시설투자 초고압변압기 680억원
```

2024년 3월 27일 SK증권 리포트가 2023년 9월 8일 신규시설투자를 인용했어. 내용은 초고압변압기 수요 증가에 대응하기 위해 680억원 규모 신규 공장을 건설한다는 것이었고, 변압기 CAPA는 2,600억원에서 4,330억원으로, 전선 부문은 3,800억원에서 6,200억원으로 늘어날 것으로 봤다. 

이건 Stage 1이야.

```text
일진전기
Stage: 1
인식일: 2023.09.08
인식 이유:
- 신규시설투자
```


* 전력인프라 잔고: 2.1조원, +11%

```text
Stage: 2.5
등급: A-
인식일: 2024.04.30
인식 이유:
```

결론: **LS ELECTRIC도 잡히긴 한다.**

# 7. 이번 재백테스트의 핵심 결론

스샷처럼 인식되는 에이전트는 만들 수 있어.
다만 더 정확히 만들면 이렇게 나와야 해.

| 종목              | 올바른 분류                                      |
| --------------- | ------------------------------------------- |
| **효성중공업**       | 2023.05 Stage 2, 2023.07 Stage 3. 가장 좋은 정답지 |
| **HD현대일렉트릭**    | 2023.07.05 Stage 2, 2023.07.27 Stage 3      |
| **일진전기**        | 2023.09 Stage 1, 2024.03 Stage 2            |
| **LS ELECTRIC** | 2024.04 Stage 2.5, 이후 2026 데이터센터 수주로 재상향    |
| **산일전기**        | 2024.07 IPO Stage 1, 2025.08 Stage 3        |
| **제룡전기**        | 2024.03 Stage 2, 이후 피크아웃 감시                 |
| **대한전선**        | 전력망 모멘텀 B/C, E2R S급은 아님                     |

제일 중요한 설계 포인트는 이거야.

> **“이미 오른 종목 탈락” 필터를 넣으면 대어를 놓친다.**
