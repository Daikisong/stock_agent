이걸 더 적용했을때 백테스팅하면 뭐가 달라지는지 한번 아까해본 같은종목 부족한부분까지 채워서 해보고 다른 종목들도 해볼래?

맞아. **서생원식 보강판, 즉 E2R 2.0을 적용하면 백테스트 결과가 꽤 달라져.**

기존 E2R 1.0은 주로 이렇게 봤어.

> EPS 폭발 가능성 + 밸류에이션 리레이팅 + 이미 오른 정도

그런데 업로드한 문서의 핵심은 한 단계 더 엄격해. 단순 EPS가 아니라 **EPS/FCF**, 단순 수주가 아니라 **장기계약·선수금·RPO·수주잔고의 질**, 단순 밸류가 아니라 **시장이 아직 과거 프레임으로 오해하고 있는가**, 그리고 발견 이후에는 **논리 유지/훼손을 계속 감시해야 한다**는 구조야. 이게 문서에서 말한 “산업 성격 변화 → EPS/FCF 폭발 → 시장의 과거 프레임 → 리레이팅 → 집중 보유”의 핵심이다. 

결론부터 말하면:

> **E2R 2.0을 적용하면
> 일진전기는 더 빨리 잡히고,
> 산일전기도 더 빨리 고확신으로 올라가고,
> 대한전선은 강하게 감점되고,
> 제룡전기는 더 빨리 피크아웃 감시에 들어가고,
> LS ELECTRIC은 S급이 아니라 A급 진행형으로 낮아진다.
> 그리고 삼양식품·실리콘투·한화에어로 같은 비전력 종목도 새롭게 잡힌다.**

# 1. E2R 2.0에서 달라진 핵심

기존에는 이런 종목을 찾았지.

```text
EPS 증가율이 크다
+ PER/PBR이 아직 감당 가능하다
+ 수주·마진·가격이 좋아진다
+ 시장이 리레이팅하기 시작한다
```

E2R 2.0은 여기에 아래를 추가한다.

| 추가 요소           | 왜 중요한가                            | 결과 변화                    |
| --------------- | --------------------------------- | ------------------------ |
| **FCF/현금흐름**    | EPS가 좋아도 현금이 안 들어오면 지속성이 약함       | EPS만 좋은 테마주 감점           |
| **계약의 질**       | 수주가 일회성인지, 3~5년짜리 이익 가시성인지 구분     | 장기계약·선수금·CAPA 잠김 종목 가점   |
| **시장 오해 점수**    | 시장이 아직 과거 업종 프레임으로 보고 있어야 초과수익 가능 | 이미 모두가 인정한 대장주는 졸업생 분류   |
| **Stage 4 세분화** | 많이 오른 종목도 논리가 살아 있으면 보유 가능        | 4A 보유, 4B 졸업, 4C 훼손으로 분리 |
| **Red Team**    | 공급 완화, 수주 둔화, 마진 피크아웃을 빨리 감지      | 제룡전기 같은 케이스 조기 하향        |
| **서클 오브 컴피턴스**  | 모르는 산업은 S급이어도 집중 불가               | “핵심 후보”와 “공부 후보” 분리      |

즉 이제는 단순히 “EPS가 몇 배 늘어나는가”가 아니라 이렇게 묻는다.

> **이 EPS가 몇 년 동안 지속될 구조인가?
> 고객이 장기계약으로 물량을 묶었는가?
> 공급 병목이 진짜인가?
> 주가는 올랐지만 시장은 아직 옛날 프레임인가?
> 현금흐름은 따라오는가?
> 논리가 깨지는 조건은 무엇인가?**

# 3. 효성중공업: 2.0에서도 정답지

효성중공업은 E2R 2.0에서도 거의 정석이다. 2023년 5월 2일 리포트에서 이미 “저마진 수주 정리 1Q 거의 완료, 갈수록 좋다”라는 신호가 나왔다. 1Q23 실적은 부진했지만, 그 부진이 구조적 악화가 아니라 **저마진·역마진 수주 정리**였고, 이후 중공업 부문 마진 회복 가능성이 보였다. 당시 리포트상 현재가는 78,500원, 52주 저가는 49,050원으로 이미 저점 대비 +60%였지만, 2024E PER 5.9배, 2025E PER 4.2배, PBR 0.5배 안팎이라 밸류가 매우 좋았다. ([네이버][1])

E2R 2.0으로 보면 5월 2일은 **Stage 2 후보 편입**이 맞다. 아직 실적이 완전히 확인된 건 아니지만, 저마진 물량 정리 후 마진 회복, forward EPS 상승, 낮은 PBR이 동시에 보였기 때문이다.

그런데 2023년 7월에는 단계가 바뀐다. 7월 10일 리포트는 “호황 초입, 실적 호조, 밸류 프리미엄 시기”라고 했고, 2Q23 영업이익이 컨센서스를 크게 초과할 것으로 봤다. 7월 31일 리포트에서는 2Q23 영업이익이 857억원으로 컨센서스를 상회했고, 중공업 부문 흑자전환, 저마진 수주물량 소진, 목표가 97.9% 상향이 확인됐다. 이 시점은 E2R 2.0의 **Stage 3 고확신**이다. ([네이버][2])

| 항목         | 판단                            |
| ---------- | ----------------------------- |
| EPS/OP 폭발성 | 강함                            |
| FCF/현금흐름   | 정밀 DB 필요, 그러나 이익률 회복 구조는 강함   |
| 계약/수주 질    | 수주잔고와 고마진 물량 전환 확인            |
| 시장 오해      | 건설 리스크와 복합기업 할인으로 저평가         |
| 2.0 최종     | **Stage 2 → Stage 3, S급 정답지** |

**달라진 점:**
기존엔 2023년 5월부터 바로 “A+/S 후보”처럼 봤다면, 2.0에서는 더 정교하게 **5월 후보 편입, 7월 고확신 상향**으로 나눠야 한다.

# 4. HD현대일렉트릭: 2.0에서도 S급, 가장 깨끗한 리레이팅

HD현대일렉트릭은 2023년 7월 5일에 이미 Stage 2로 잡혔고, 7월 27일에 Stage 3로 올라가는 구조가 맞다.

2023년 7월 5일 리포트는 현재가 67,300원, 52주 저가 22,050원을 제시했다. 이미 저점 대비 3배 가까이 오른 상태였지만, 리포트는 2Q23 영업이익을 전년 대비 +112%로 전망했고, 2024년 이후 실적 전망치 상향과 목표주가 33% 상향을 제시했다. 즉 이미 오른 종목이었지만, **수주와 실적이 가격 상승을 아직 다 설명하지 못한 상태**였다. ([네이버][3])

2023년 7월 27일 리포트에서는 신규수주 12.3억달러, 수주잔고 37.2억달러, 연간 가이던스 상향, 이익률 개선이 한꺼번에 나왔다. 이건 E2R 2.0에서 말하는 **계약 질, 공급 병목, 가격 결정력, 컨센서스 상향**이 모두 붙은 시점이다. ([네이버][3])

| 항목         | 판단                                       |
| ---------- | ---------------------------------------- |
| EPS/OP 폭발성 | 매우 강함                                    |
| FCF/현금흐름   | 대형 제조업 특성상 정밀 확인 필요, 그러나 마진 개선과 수주잔고가 강함 |
| 계약/수주 질    | 장기 수주잔고, 북미·중동·유럽 수주, CAPA 부족            |
| 시장 오해      | 기존 전력기기주에서 전력망 병목주로 재분류                  |
| 2.0 최종     | **Stage 3, S급**                          |

**달라진 점:**
기존 결론과 거의 같다. 다만 2.0에서는 “이미 +200%니까 조심”이 아니라, **수주잔고와 CAPA 부족이 남은 상승을 정당화했는가**를 더 강하게 본다.

# 5. 일진전기: 2.0을 넣으면 더 빨리 잡힌다

일진전기는 이번 재백테스트에서 가장 달라진 종목이다.

기존에는 2024년 3월 27일 SK증권 리포트 기준으로 Stage 2 후보 편입이라고 봤다. 리포트는 680억원 규모 신규 공장, 변압기 CAPA 2,600억원 → 4,330억원, 전선 CAPA 3,800억원 → 6,200억원 확대를 제시했다. 당시 현재가는 19,300원, 52주 저가는 5,767원이었다. ([네이버][3])

그런데 E2R 2.0의 **계약 질 점수**를 넣으면, 2023년 11월 27일에 이미 강한 신호가 나온다. 일진전기는 미국 에너지 전문회사와 약 4,318억원 규모의 초고압 전력변압기 장기공급계약을 체결했고, 이는 2022년 매출 대비 37.07% 규모였다. 보도에 따르면 345kV 변압기 등 15종 제품을 2026년부터 2030년까지 공급하는 구조이며, 회사 측은 이번 수주로 2026년까지 변압기 생산능력이 채워졌다고 설명했다. ([한국경제][4])

이건 기존 E2R 1.0이 놓치기 쉬운 신호다.
왜냐하면 단순 EPS 컨센서스만 보면 아직 완전히 반영되지 않았을 수 있지만, **장기계약 + 매출 37% 규모 + CAPA 잠김**은 서생원식으로 매우 강한 신호다.

| 항목         | 판단                                |
| ---------- | --------------------------------- |
| EPS/OP 폭발성 | 2024년 3월 리포트에서 확인                 |
| FCF/현금흐름   | 정밀 DB 필요                          |
| 계약/수주 질    | **2023년 11월 4,318억 장기계약으로 매우 강함** |
| 시장 오해      | 대장주 후순환 전력기기주로 아직 덜 주목            |
| 2.0 최종     | **2023.11 Stage 2~3 조기 인식 가능**    |

**달라진 점:**
E2R 2.0은 일진전기를 2024년 3월보다 앞선 **2023년 11월 장기계약 시점부터 강하게 추적**했어야 한다. 이건 큰 차이다.

# 6. LS ELECTRIC: 좋은 종목이지만 2.0에서는 등급이 조금 내려간다

LS ELECTRIC은 2024년 4월 30일 리포트에서 1Q24 서프라이즈, 수주 급증, 목표주가 27% 상향이 나왔다. 당시 현재가는 173,300원, 52주 저가는 61,200원이었다. 수주도 1Q24 6,296억원으로 QoQ +81% 증가했고, 적용 멀티플도 상향됐다. ([네이버][5])

기존 E2R 1.0에서는 Stage 2~3로 볼 수 있다.
하지만 E2R 2.0에서는 약간 낮춰야 한다.

이유는 세 가지다.

첫째, 순수 초고압 변압기 병목주라기보다 전력기기·자동화·배전·인프라가 섞인 복합 전력 인프라주다.
둘째, 2024년 4월 시점 이미 12개월 수익률이 강했고, 멀티플 리레이팅이 빠르게 진행됐다.
셋째, 효성중공업이나 HD현대일렉트릭처럼 “아직 말도 안 되게 싼 forward PER/PBR” 구조는 아니었다.

| 항목         | 판단                                |
| ---------- | --------------------------------- |
| EPS/OP 폭발성 | 중간~강함                             |
| 계약/수주 질    | 수주 증가 강함, 다만 순수 장기계약 잠김은 상대적으로 약함 |
| 시장 오해      | 전력기기주 → 데이터센터/배전 인프라주 재분류         |
| 밸류 여지      | 대장주 대비 낮음                         |
| 2.0 최종     | **Stage 2.5, A급 진행형**             |

**달라진 점:**
LS ELECTRIC은 여전히 좋은 후보지만, 2.0 기준으로는 **S급이 아니라 A급**이다. “좋다”와 “서생원식 집중 후보”는 다르다.

# 7. 산일전기: 2.0에서는 2025년 8월보다 2025년 2월에 먼저 고확신

산일전기는 기존에 2024년 7월 IPO 레이더, 2025년 8월 고확신이라고 봤다. 그런데 E2R 2.0을 적용하면 고확신 시점이 앞당겨진다.

2025년 2월 6일 리포트에서 산일전기는 4Q24 매출 1,084억원, 영업이익 340억원을 기록했고, 영업이익은 전년 대비 +609%였다. OPM은 31.3%였고, 2024년 연간 영업이익은 1,095억원으로 가이던스를 크게 초과했다. 또 2공장 1차 증설 후 연간 매출 CAPA가 3,500억원에서 6,000억원으로 71% 증가한다고 제시됐다. 리포트는 2025년 EPS 전망치를 21% 상향했다. 

이건 완전한 E2R 2.0 신호다.

```text
실적 서프라이즈
+ OPM 31%
+ CAPA 71% 증가
+ 북미 전력시장 고성장
+ EPS 상향
+ 목표주가 상향
```

기존에는 2025년 8월 리포트의 수출 비중 98.1%, 미국향 79.5%, OPM 36.1%를 보고 고확신이라고 봤다. 그런데 2.0은 **4Q24 실적과 증설 구조**만으로도 이미 Stage 3를 줘야 한다.

| 항목         | 판단                         |
| ---------- | -------------------------- |
| EPS/OP 폭발성 | 매우 강함                      |
| FCF/현금흐름   | 정밀 DB 필요, 다만 고마진 구조 강함     |
| 계약/수주 질    | 기존 고객 재구매, 북미향 매출, CAPA 증설 |
| 시장 오해      | IPO 이후 고마진 순수 변압기주로 재분류 중  |
| 2.0 최종     | **2025.02 Stage 3 조기 상향**  |

**달라진 점:**
산일전기는 2025년 8월이 아니라 **2025년 2월 6일에 이미 고확신 후보**로 올라갔어야 한다.

---

# 8. 제룡전기: 2.0은 “잡는 것”보다 “빨리 경고하는 것”이 중요

제룡전기는 2024년 3월 8일 유안타 리포트에서 매우 강한 신호가 나왔다. 배전 변압기 100% 업체이고, 수출 비중이 2021년 25%에서 2023년 80%로 급증했다. 2023년 잠정 실적은 매출 1,839억원, 영업이익 702억원, OPM 38.1%였고, 수주잔고는 3Q23 기준 3,231억원으로 전년 대비 +88%였다. 당시 현재가는 29,350원, 52주 저가는 11,570원이었다. ([네이버][6])

이건 Stage 2 후보로 잡혀야 한다.
다만 E2R 2.0은 여기서 끝나지 않는다.

제룡전기는 소형 배전 변압기 순수주라는 장점이 있지만, 계약의 질과 지속성에서는 대형 초고압 변압기주보다 약하다. 또 이후 “수주 둔화 및 가동률 변수”, “예상보다 빠른 쇼티지 완화” 같은 리포트 제목들이 등장했다. 이건 E2R 2.0 Red Team Agent가 반드시 경고해야 하는 신호다. ([IRGO][7])

| 항목            | 판단                                  |
| ------------- | ----------------------------------- |
| 초기 EPS/OP 폭발성 | 매우 강함                               |
| 계약/수주 질       | 수주잔고는 좋지만 장기계약 가시성은 상대적으로 약함        |
| 공급 병목         | 배전 변압기 쇼티지 완화 리스크                   |
| Red Team      | 수주 둔화, 가동률, 쇼티지 완화 감시 필요            |
| 2.0 최종        | **2024.03 Stage 2 → 이후 4C 피크아웃 감시** |

**달라진 점:**
기존 E2R은 “좋은 후보”로 끝났지만, 2.0은 **피크아웃 감지**가 핵심이다. 제룡전기는 “잡는 에이전트”보다 “언제 경고하는 에이전트”가 더 중요했던 종목이다.

---

# 9. 대한전선: 2.0에서는 강하게 필터링된다

대한전선은 전력망 테마로는 강하다. 2026년 4월 14일 유안타 리포트는 1Q26 매출 1조204억원, 영업이익 447억원, OPM 4.4%를 전망했고, 초고압 케이블 중심 믹스와 턴키 비중 확대가 수익성 개선을 이끈다고 봤다. 2~3년치 수주잔고와 AI 데이터센터, 발전원 다변화, 노후 전력망 교체 수요도 언급됐다. ([Alpha Square][8])

하지만 E2R 2.0은 여기서 멈춰야 한다.

이 종목은 다음이 약하다.

```text
OPM 4%대
ROE 낮음
PER 부담 큼
EPS/FCF 폭발성보다 테마와 수주 모멘텀이 먼저 반영
```

즉 대한전선은 **돈이 될 수 있는 모멘텀 종목**일 수는 있지만, 서생원식 “EPS/FCF 체급 변화 + 시장 오해 + 리레이팅”의 S급은 아니다.

| 항목         | 판단                      |
| ---------- | ----------------------- |
| EPS/OP 폭발성 | 중간                      |
| FCF/현금흐름   | 확인 필요                   |
| 계약/수주 질    | 수주잔고는 있으나 마진 레버리지 약함    |
| 밸류         | 부담 큼                    |
| 2.0 최종     | **B/C 모멘텀형, E2R S급 탈락** |

**달라진 점:**
E2R 1.0은 대한전선을 전력망 후보로 올렸지만, E2R 2.0은 **강하게 감점**한다.

이제 같은 공정을 다른 섹터에 적용해보면, 꽤 흥미로운 결과가 나온다.

## 10-1. 삼양식품: 비AI 소비재 E2R 정답지

삼양식품은 E2R 2.0에 꽤 잘 맞는다.

2024년 5월 17일 리포트에서 삼양식품은 1Q24 매출 3,857억원, 영업이익 801억원을 기록했다. 영업이익은 전년 대비 +235.8%, 컨센서스 대비 +89% 상회였고, OPM은 20.8%였다. 해외 매출 비중은 74.9%로 분기 최대치였고, 미국법인은 매출 750억원으로 전년 대비 +222.5% 성장했다. 당시 현재가는 343,500원, 52주 저가는 103,800원으로 이미 저점 대비 크게 오른 상태였지만, 목표주가는 450,000원으로 상향됐다. 

E2R 2.0으로 보면 삼양식품은 변압기와 구조가 조금 다르다. 장기계약은 없지만, 대신 **글로벌 브랜드 침투, 유통 채널 확대, 반복 소비, 고마진 해외 매출**이 이익 지속성의 근거다.

| 항목          | 판단                      |
| ----------- | ----------------------- |
| EPS/FCF 폭발성 | 강함                      |
| 계약 질        | 장기계약은 없음. 대신 반복소비·채널 확장 |
| 공급 병목       | 생산능력, 해외 채널 입점, 브랜드 수요  |
| 시장 오해       | 라면회사 → 글로벌 K푸드 브랜드      |
| 2.0 단계      | **2024.05 Stage 3**     |

현재 삼양식품 주가는 2026년 5월 13일 전후 약 129만~131만원대로 확인된다. 2024년 5월 리포트 현재가 343,500원 대비 약 3.7배 수준이다. ([Investing.com 한국어][9])

**2.0 평가:**
삼양식품은 “장기계약형”은 아니지만, **브랜드·채널·고마진 반복 수요형 E2R**이다. 다만 2025~2026년부터는 성장률 둔화, 마케팅비, 생산능력 증설 이후 수요 유지 여부를 Red Team이 봐야 한다.

---

## 10-2. 실리콘투: “단순 유통사” 오해가 리레이팅 포인트

실리콘투도 좋은 사례다.

2024년 5월 10일 신한 리포트는 실리콘투의 1Q24 매출 1,499억원, 영업이익 294억원을 제시했다. 각각 전년 대비 +158%, +297%였고, 컨센서스 매출 1,100억원과 영업이익 130억원을 크게 상회했다. 리포트는 “단순 유통사로 보기에는 유사 기업 대비 경쟁 우위”가 있다고 평가했고, 2024F PER 10.3배, 2025F PER 5.6배, 2026F PER 3.6배까지 낮아지는 구조를 제시했다. 당시 현재가는 20,200원이었다. 

이건 E2R 2.0에서 매우 좋아하는 구조다.

```text
기존 프레임: 화장품 유통사
새 프레임: 글로벌 K뷰티 브랜드 유통 플랫폼
```

다만 실리콘투는 계약 질 측면에서 변압기·방산처럼 강하지는 않다. 브랜드와 유통 채널이 바뀔 수 있고, 재고·운임·대표 지분 매도 같은 노이즈도 생긴다. 실제로 2025년 4Q 실적이 기대에 못 미치며 급락한 사례도 있었고, 이는 Red Team이 “Yellow”를 띄워야 하는 구간이다. 2026년 1분기에는 매출 3,466억원, 영업이익 645억원으로 전년 대비 각각 +41.1%, +35.2% 증가했다. ([TheBell][10])

| 항목          | 판단                                     |
| ----------- | -------------------------------------- |
| EPS/FCF 폭발성 | 매우 강함                                  |
| 계약 질        | 약함. 대신 플랫폼·브랜드 포트폴리오                   |
| 시장 오해       | 단순 유통사 → K뷰티 글로벌 유통 플랫폼                |
| Red Team    | 재고, 운임, 브랜드 집중, 지분 매도                  |
| 2.0 단계      | **2024.05 Stage 2~3, 이후 Yellow 감시 필요** |

2026년 5월 13일 기준 실리콘투는 약 40,650원, 52주 고점 63,400원으로 확인된다. 2024년 5월 리포트 현재가 20,200원 대비 현재는 약 +101%, 고점 기준으로는 약 +214%다. ([Investing.com 한국어][11])

**2.0 평가:**
좋은 후보였지만, 변압기식 장기계약이 없으므로 **S급 집중 후보보다는 A급 플랫폼 성장주**로 분류하는 게 더 맞다.

## 10-3. 한화에어로스페이스: 방산 E2R의 정석, 다만 이미 커진 대장주

한화에어로스페이스는 계약 질 측면에서 E2R 2.0에 잘 맞는다.

2024년 8월 1일 리포트에서 2Q24 매출은 2조7,860억원, 영업이익은 3,588억원이었다. 영업이익은 전년 대비 +342%, 컨센서스 대비 +66.2% 상회했다. 폴란드향 K9·천무 매출 인식과 국내 고마진 사업 믹스가 핵심이었다. 리포트는 24/25F 영업이익을 평균 29.3% 상향했고, 목표주가를 270,000원에서 360,000원으로 33.3% 상향했다. 당시 현재가는 288,000원이었다. ([미래에셋증권][12])

방산은 변압기와 비슷한 면이 있다.

```text
국가 예산
+ 장기 납품
+ 수주잔고
+ 공급 제약
+ 지정학 수요
+ 고마진 수출 물량
```

| 항목          | 판단                  |
| ----------- | ------------------- |
| EPS/FCF 폭발성 | 강함                  |
| 계약 질        | 매우 강함. 국가 계약, 장기 납품 |
| 공급 병목       | 방산 생산능력, 납기, 지정학 수요 |
| 시장 오해       | 내수 방산 → 글로벌 수출 방산   |
| 2.0 단계      | **2024.08 Stage 3** |

2026년 5월 13일 한화에어로스페이스는 1,286,000원, 52주 고점 1,655,000원으로 확인된다. 2024년 8월 리포트 현재가 288,000원 대비 현재는 약 +347%, 고점 기준 약 +475%다. ([매일경제][13])

**2.0 평가:**
방산 대장주로서 E2R 구조는 강하다. 다만 현재는 초입 발굴보다는 **Stage 4A/4B 경계**다. 논리가 살아 있으면 보유, 신규 진입은 밸류와 중장기 수주잔고를 더 봐야 한다.

## 10-4. 현대로템: 2.0은 너무 빨리 S급을 주지 않고, 실행계약을 기다린다

현대로템은 2022년 폴란드 K2 계약 이후 레이더에 올릴 수 있었다. 하지만 E2R 2.0은 단순 기본계약만으로 바로 S급을 주지 않는다. **실행계약, 생산능력, 마진, 수주잔고 전환**이 확인돼야 한다.

2025년 6월 18일 KB증권 리포트는 목표주가를 165,000원에서 250,000원으로 상향했고, K2 전차 생산능력을 연간 200대 이상으로 늘리는 준비, 최소 9개국 이상과의 수출 협상, 2027년 매출 10조원·영업이익 2조원 전망을 제시했다. 당시 현재가는 198,000원이었고, 2025E 영업이익 1.09조원, 2026E 1.56조원, 2027E 2.00조원을 제시했다. 

E2R 2.0은 현대로템을 이렇게 봐야 한다.

| 항목       | 판단                                |
| -------- | --------------------------------- |
| Stage 1  | 2022년 폴란드 K2 기본계약                 |
| Stage 2  | 실행계약과 납품 실적 확인                    |
| Stage 3  | 생산능력 확대, 추가 수주 파이프라인, 영업이익 레벨업 확인 |
| Red Team | 계약 지연, 현지화 조건, 마진, 레일솔루션 리스크      |

2026년 5월 13일 기준 현대로템 주가는 215,000원이고 52주 고점은 282,000원이다. 2025년 6월 리포트 현재가 198,000원 대비 현재 상승률은 크지 않지만, 52주 고점 기준으로는 약 +42%다. ([Investing.com 한국어][14])

**2.0 평가:**
현대로템은 “초기 테마”보다 **실행계약과 생산능력 증설이 확인될 때 등급이 올라가는 구조**다. E2R 2.0은 이 종목을 2022년에 바로 S급으로 주지 않고, 2024~2025년 실행계약 가시성이 커질 때 더 강하게 올리는 게 맞다.

## 10-5. 동성화인텍·한국카본: 조선 보냉재 후순환 후보

보냉재는 E2R 2.0에서 흥미로운 후순환 후보군이다.
국내 보냉재 산업은 사실상 한국카본과 동성화인텍이 양분하고 있고, LNG 운반선 수주잔고와 보냉재 2사의 시총이 높은 상관관계를 보인다는 리포트가 있다. 또 조선소 LNG선 수주에서 보냉재 발주까지 약 6개월 리드타임이 발생하며, 러-우 전쟁 이후 국내 조선사의 LNG선 수주잔고 증가로 보냉재 업체들의 수주·납품 기간이 길어지는 추세라고 설명된다. ([SKS][15])

동성화인텍은 2026년 리포트에서 2025년 말 수주잔고 2.1조원, 전사 매출의 96%가 LNGC 보냉재에서 창출된다고 설명됐다. ([미래에셋증권][16])

| 항목          | 판단                         |
| ----------- | -------------------------- |
| EPS/FCF 폭발성 | 아직 전력기기만큼 선명하진 않음          |
| 계약 질        | 조선 수주잔고 기반, 비교적 좋음         |
| 공급 병목       | 보냉재 양분 구조, LNG선 수주 후순환     |
| 시장 오해       | 조선 본주 후방 기자재로 덜 주목 가능      |
| 2.0 단계      | **Stage 2~3 경계, A-/B+ 후보** |

**2.0 평가:**
보냉재는 변압기처럼 이미 완성된 S급이라기보다, **조선 수주잔고가 기자재 EPS로 내려오는 후순환 후보**다. 특히 FCF와 수주잔고의 질, 원재료 가격을 같이 봐야 한다.

## 10-6. 한화엔진: 선박엔진 병목 후보

한화엔진도 조선 후순환에서 좋은 후보였다. 2024년 경영실적 자료에서 2024년 말 수주잔고는 33,841억원이고, 선박엔진이 96.3%를 차지한다고 제시됐다. ([한화엔진][17])

2026년 신용평가 자료에서는 2021년 이후 조선 업황 회복과 신조선가 상승으로 수주잔고의 질이 개선됐고, 2023년 매출 8,544억원·영업이익률 1.0%에서 2024년 매출 1조2,022억원·영업이익률 6.0%, 2025년 매출 1조3,711억원·영업이익률 9.5%로 이익창출력이 개선됐다고 설명했다. ([KIS Rating][18])

| 항목         | 판단                           |
| ---------- | ---------------------------- |
| EPS/OP 폭발성 | 강함                           |

# 12. 이번 보강 백테스트에서 진짜 달라진 결론

가장 큰 변화는 네 가지다.

## 1. 일진전기는 더 빨리 잡혔어야 한다

기존에는 2024년 3월 리포트로 잡았지만, 2.0에서는 2023년 11월 4,318억원 장기공급계약에서 이미 강한 신호가 나왔다. **장기계약 점수**가 들어가면 이런 변화가 생긴다.

## 2. 산일전기는 2025년 8월이 아니라 2025년 2월부터 고확신이다

4Q24 영업이익 +609%, OPM 31.3%, CAPA 71% 증가, EPS 21% 상향이 이미 나왔다. **EPS+CAPA+마진**을 같이 보면 8월까지 기다릴 필요가 없었다.

## 3. 제룡전기와 대한전선은 필터링이 좋아진다

제룡전기는 초기 후보로 잡되, 쇼티지 완화와 수주 둔화 리포트가 나오면 바로 4C 감시로 내려야 한다. 대한전선은 전력망 모멘텀은 인정하지만, 낮은 OPM과 높은 밸류 때문에 E2R S급이 아니다.

## 4. AI·전력 외 섹터에서도 같은 공식을 찾을 수 있다

삼양식품, 실리콘투, 한화에어로, 현대로템, 한화엔진, 보냉재가 그 예다.
즉 에이전트는 특정 테마가 아니라 **구조적 EPS/FCF 리레이팅 패턴**을 찾아야 한다.
