## 6. Residual Finding
이번 holdout의 핵심은 **trust event의 고장 위치**를 나누는 것이다. 같은 “조작·불공정거래·임원 매도·거래정지”라도, 가격의 엔진룸이 완전히 깨진 경우와 이미 충격이 가격에 반영된 뒤 reviewable overhang만 남은 경우는 다르다. 전자는 hard 4C, 후자는 Stage4B-Watch/false-break audit가 맞다.

- `카카오 2023/2024`와 `파두 2023-11-14`, `대양금속 2023-10-26`은 legal/trust smoke가 있었지만 hard 4C를 바로 고정하면 이후 MFE를 놓친다. 특히 카카오 2023 row는 MFE90 56.31%, MAE90 -5.81%로 reviewable-event 성격이 강했다.
- `카카오페이 2022`, `SG-CFD cluster`, `영풍제지 2023`은 trust event가 issuer price formation을 실제로 망가뜨렸다. 카카오페이는 MFE90 6.69% / MAE90 -43.14%, 영풍제지는 MFE180 0.00% / MAE180 -94.14%로 hard 4C가 맞았다.
- `대양금속`은 같은 영풍제지 linkage라도 parent/affiliate linkage와 직접 조작 unwind를 분리해야 한다. MAE90 -32.51%는 위험하지만 MFE180 68.25%가 열려 hard 4C 고정 대신 4B-Watch가 더 정확했다.

## 7. Candidate Shadow Rule
```text
rule_id: R13_ACCOUNTING_TRUST_REVIEWABLE_VS_IRREVERSIBLE_PRICE_VALIDATION_GATE_V2
scope: R13_CROSS_ARCHETYPE_ACCOUNTING_TRUST_PRICE_VALIDATION
proposal_type: shadow_rule_candidate_only
production_scoring_changed: false
```
Rule statement:

1. **Irreversible trust break**: confirmed manipulation unwind, executive cash-out that destroys minority alignment, trading-resumption collapse, or direct issuer fraud/false disclosure with no reset-entry support -> Stage4C or hard Stage4B/4C protection.
2. **Reviewable trust overhang**: legal investigation, arrest/indictment, proxy linkage, post-shock reset, or parent/affiliate contamination without direct issuer cash-flow break -> Stage4B-Watch first, with price-validation and later legal/vote/court gate.
3. If `MFE90 >= 35` and `MAE90 > -25`, hard 4C should require a second irreversible confirmation. If `MFE90 < 15` and `MAE90 <= -30`, hard trust penalty is reinforced. If both MFE and MAE are high, use staged-entry/local-4B rather than clean promotion.
