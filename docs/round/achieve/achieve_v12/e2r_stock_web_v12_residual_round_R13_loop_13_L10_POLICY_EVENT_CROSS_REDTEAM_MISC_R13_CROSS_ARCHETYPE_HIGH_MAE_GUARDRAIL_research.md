## 3. Mechanism Diagnosis

High MAE is the market’s undertow. Early MFE is the wave the profile sees first; deep MAE is the current underneath. The current calibrated profile already blocks many price-only blowoffs, but source-proxy rows can still look better than they are if only the upside is remembered.

The guardrail should split rows into four buckets:

- **controlled_MAE_positive**: positive path is plausible, but evidence must be refreshed before Green/full4B.
- **high_MAE_promotion_block**: MFE exists, but MAE path requires cap until non-price bridge is refreshed.
- **hard_MAE_guard**: MAE180 is too deep for promotion unless a fresh bridge materially changes the thesis.
- **hard_4C / narrative block**: thesis break, binary failure, PF/workout, or discontinuity requires 4C watch or blocked calibration.
