## 4. 4B / 4C decision grammar

```text
Stage4B-Watch / local 4B
  = event cap, late entry, qualification-only, bridge exhaustion, proxy blowoff, or price heat.
  = Do not promote to Stage3/Green; use staged entry, position-size, and exit guard.

Stage4C / Stage4C-Watch
  = issuer-level economics, order, utilization, direct production, customer call-off, or thesis axis actually breaks.
  = Hard 4C requires second non-price confirmation.

False-hard-4C audit
  = headline looks broken but price was already reset, forward MFE reopens, or evidence is not issuer-irreversible.
  = Keep as Stage4C-Watch / false-break audit rather than immediate hard 4C.

Control positive
  = hard bridge + conversion bridge + favorable path.
  = Preserve Stage2-Actionable/Yellow; do not let generic 4B/4C guard overblock.
```
