## 13. Current Calibrated Profile Stress Test

The current calibrated profile is correct when C16 evidence contains a measurable bridge. It is still vulnerable to two residual errors: `(1)` policy/resource labels that are actually proxy spikes and `(2)` structural resource bridge cases that are under-recognized until the price has already moved.

## 14. Stage2 / Yellow / Green Comparison

Stage2-Actionable worked for POSCO홀딩스 and LX인터내셔널 when the trigger represented resource project/trading bridge rather than price-only theme chasing. Stage3-Yellow worked for LS only when copper supply and grid/resource bridge were both present. No Stage3-Green trigger is proposed in this loop.

## 15. 4B Local vs Full-window Timing Audit

C16 needs a local 4B watch layer. For rare-earth and copper alloy proxies, local 4B should block promotion because MFE was small while MAE90/180 became deep. For 금양, the row is more subtle: MFE30 was high, but MAE90/180 later became severe, so the rule should not call it a clean positive without risk overlay.

## 16. 4C Protection Audit

유니온 and 서원 are thesis-break watch examples, but not automatic hard 4C. The proposed C16 rule weakens generic hard-4C routing when only the policy/commodity label fades; it requires issuer-level supply bridge break, financing stress, project cancellation, or customer-pull evidence.
