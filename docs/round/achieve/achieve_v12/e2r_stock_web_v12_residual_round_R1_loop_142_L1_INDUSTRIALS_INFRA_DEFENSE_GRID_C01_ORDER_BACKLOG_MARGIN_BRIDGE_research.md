## 4. Research Thesis

C01 should not treat “수주잔고가 많다” as a free Stage2-Actionable pass. The useful C01 bridge is narrower: **signed order or visible backlog → delivery/revenue timing → margin/revision/FCF conversion**. The opposite trap is also important. Some order-cycle names move before reported margin turns, while some high-margin or earnings-beat names fail because volume, demand, or post-peak exit risk is worse than the headline.

This loop tests a second C01 leaf set after the shipbuilding-heavy loop132: LNG insulation, ship engines, industrial fittings, high-margin valves, construction equipment, and compact equipment demand decay.

## 9. Shadow Rule Candidate

```text
sector_specific_rule_candidate = L1_C01_INDUSTRIAL_BACKLOG_MARGIN_CONVERSION_AND_EXIT_GATE_V2
canonical_archetype_rule_candidate = C01_ORDER_BACKLOG_MARGIN_TIMING_AND_DEMAND_DECAY_GATE_V2
```

### Proposed rule language

For C01, Stage2-Actionable requires at least two of the following: `signed order / visible backlog`, `delivery or revenue recognition timing`, `margin or ASP bridge`, `customer/project quality`, `revision or FCF visibility`. If near-term margin is still weak but order intake is reaccelerating and revenue recognition cycle is short, allow Stage2-Watch / early Stage2 rather than blocking. If reported margin is strong but volume/order growth is absent, or if the first earnings-beat peak reverses into MAE90 <= -20%, require local 4B/exit guard.
