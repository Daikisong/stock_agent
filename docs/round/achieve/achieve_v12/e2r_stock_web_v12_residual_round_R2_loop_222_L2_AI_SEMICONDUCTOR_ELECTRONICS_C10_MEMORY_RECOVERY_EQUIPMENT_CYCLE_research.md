## 1. Why this loop exists

C10 rows are a magnet for two opposite mistakes. One side treats every HBM/CXL sentence as if it were a booked equipment order. The other side is too conservative when a small supplier has already crossed a customer-qualification or reset-asymmetry threshold. This loop separates the pipe from the water: memory-cycle weather, issuer product route, customer qualification, booked order/shipment, margin conversion, and price phase.

The No-Repeat Index already has a large corpus, so the target is not row count. The target is quality: direct URLs, non-duplicate symbols/families, complete 30D/90D/180D MFE-MAE rows, and cases where the current calibrated profile still confuses proxy evidence with conversion evidence.

## 5. Case notes

### C10-L222-001 — EO Technics / HBM laser-annealing expectation
The evidence was real enough to justify a watch row, but not enough for conversion credit. The path behaved like a late proxy: 9.34% 180D MFE versus -55.84% 180D MAE. The shadow rule should cap generic HBM equipment expectation unless a customer/order or margin bridge is visible.

### C10-L222-002 — Wonik IPS / front-end memory capex proxy
The source itself contained the caveat: HBM alone was not enough; broader DRAM/NAND investment and product-mix fit mattered. The price row confirmed the caveat: 9.96% MFE against -42.97% 180D MAE.

### C10-L222-003 — Micro Contact Solution / reset socket route
This is the missed-structural side. A package-test socket route after a reset produced 417.46% 180D MFE with only -10.16% MAE. It still should not be Green from profile evidence alone; the rule should allow Actionable, then wait for booked order or margin conversion.

### C10-L222-004 — Micro2Nano / SK hynix HBM EDS probe-card qualification
Customer qualification is stronger than generic HBM proxy. The result was positive, but -23.03% early MAE demands a size/phase guard. Qualification can open Stage2-Actionable; Yellow/Green still require actual order/ramp and margin conversion.

### C10-L222-005 — Intekplus / HBM package inspection value-chain proxy
The sector value-chain map created a tradable early pop, but the issuer row did not hold: -58.63% 180D MAE after the March peak. This is a clean proxy-cap counterexample.

### C10-L222-006 — TES / DRAM/HBM equipment-lineup transition after reset
TES is the constructive opposite of the proxy failures. The November 2024 reset and DRAM/HBM-linked equipment-lineup transition produced 100.40% 180D MFE with -11.67% MAE. Yellow is permitted; Green waits for booked order and margin confirmation.
