## 10. Residual Interpretation

The main residual is not that the current profile lacks a 4B/4C rule. It already has one. The residual is **door selection**.

- **Local 4B-watch** is a yellow traffic light: it slows the model when evidence is stale, result-only, price-phase late, or policy-premium heavy.
- **Full 4B** needs non-price weakness: cost-rate damage, cash conversion failure, utilization deterioration, explicit event cap, or missing project cashflow.
- **Hard 4C** is not a bad quarter. It is a broken engine: repeated thesis damage, demand collapse, legal/customer/project termination, or financing/liquidity break.

The counterexample rows are the places where the sign on the road said “Green,” but the bridge had already started to rust. The anti-overfit row is the opposite warning: do not close the bridge at the first vibration when traffic is still genuinely crossing.
