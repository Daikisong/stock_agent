## 1. Selection Rationale
The main execution prompt requires `coverage_index_first`, not a mechanical R1→R13 sequence. After the same-session local fills and the prior R13 `4B/4C` and `Stage2 false-positive` checkpoints, the remaining R13 scope with a distinct mechanism is `R13_CROSS_ARCHETYPE_ACCOUNTING_TRUST_PRICE_VALIDATION`.

The No-Repeat Index lists this R13 scope as an already-covered but still valid quality-repair checkpoint. Therefore this loop does not claim a new sector unlock. It asks a narrower cross-sector question:

> When a row looks Actionable/Yellow, does the bridge survive accounting, reserve, CET1, PF balance-sheet, clinical-data, corporate-action, and price-path validation?

Positive controls keep eligibility when a verified bridge exists. Counterexamples route to local 4B or hard 4C when the bridge is only a label, or when trust/financing/construction quality/corporate split context breaks the thesis.

## 7. 4B / 4C Audit
This loop is not a general 4B proof. The new distinction is whether the price path should be trusted after accounting/trust validation. A row can be strong on relative strength and still fail if the accounting or price-validation context is not clean. Conversely, a row with accounting-heavy evidence can be valid if the bridge is explicit and the drawdown path stays controlled.

- Full 4B remains blocked when evidence is price-only.
- Local 4B/watch is appropriate when the row has label strength but lacks accounting/trust bridge.
- Hard 4C should be allowed earlier when issuer-level construction quality, financing trust, or corporate split price-validation breaks the original thesis.
- Verified reserve/CET1/PF/data bridge rows should not be forced into 4B only because the broad sector has past trust problems.

## 8. Canonical-Archetype Rule Candidate
`R13_accounting_trust_price_validation_bridge_gate`:

Stage2/Stage2-Actionable/Stage3-Yellow eligibility remains valid only if at least one of the following bridge families is verified:

```text
- CSM / K-ICS / reserve / loss-ratio quality and capital-return bridge
- ROE / CET1 / actual dividend or buyback bridge
- PF balance-sheet / refinancing / cash-flow bridge
- clinical endpoint / partner / commercialization bridge
- corporate-action-clean price path and non-event margin bridge
```

Rows lacking those bridges should remain `Stage1-Watch` or `local_4B_watch`. Rows with issuer-level trust break should be eligible for `Stage4C` before waiting for formal/legal closure.
