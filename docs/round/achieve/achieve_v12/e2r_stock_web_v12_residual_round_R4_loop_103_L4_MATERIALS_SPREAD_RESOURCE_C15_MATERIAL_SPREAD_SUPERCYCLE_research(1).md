## 4. Current calibrated profile stress test

The current calibrated proxy already blocks pure price-only blowoff and requires non-price evidence for full 4B. C15 still has a residual problem: a materials headline often looks like a broad macro confirmation while the company-level bridge is missing. This loop stresses that boundary.

- **Positive exception:** 풍산 is usable because copper exposure is paired with a defense/backlog margin bridge; it should not be generalized to the entire material sector.
- **Wrong-archetype contaminant:** 고려아연 after September 2024 is not clean C15. The path is dominated by governance/control-premium/tender dynamics and should be capped or rerouted toward C32.
- **False-positive cluster:** POSCO홀딩스, 대한제강, 롯데케미칼, and S-Oil show that commodity-label recovery can create short bounces but not durable 90D/180D alignment without ASP/volume/OPM/FCF conversion.
