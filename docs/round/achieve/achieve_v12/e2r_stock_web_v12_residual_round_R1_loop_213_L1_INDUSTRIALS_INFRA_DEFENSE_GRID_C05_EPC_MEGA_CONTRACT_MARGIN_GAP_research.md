## 1. Round selection and no-repeat rationale

No-Repeat Index 기준으로 C01~C32는 모두 80 row 이상이라 이번 실행은 신규 row 수 채우기가 아니라 품질 보강이다. 직전 산출물이 C13이므로 즉시 반복을 피했고, Priority 1의 C05 — `margin/working-capital 실패 반례와 4C 전환 타이밍 보강` — 을 선택했다. 로컬 `/mnt/data` 내 C05 산출물은 loop 208~212까지 확인되어 이번 파일은 C05 loop 213으로 배정했다.
