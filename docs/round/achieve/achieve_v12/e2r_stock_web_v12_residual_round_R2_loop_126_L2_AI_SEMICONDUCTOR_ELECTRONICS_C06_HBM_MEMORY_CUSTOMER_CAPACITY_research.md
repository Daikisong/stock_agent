## 4. Residual interpretation

### 4.1 SK hynix: leader confirmation is not always a fresh entry

SK hynix is the cleanest C06 name when customer allocation, HBM mix, product generation and capacity discipline all line up. But the same company can create two very different calibration lessons. On 2024-07-24, the evidence confirmed leadership, 12-layer HBM3E shipment plans and potential DRAM-profit contribution, yet the stock was already crowded. The path then produced `MFE_90D=1.92%` and `MAE_90D=-30.60%`, so this is a local 4B guard, not fresh Yellow.

By contrast, 2024-11-27 and 2025-01-24 are cleaner Actionable examples. The first adds capex discipline, shareholder return and financial-soundness targets to the HBM leadership story. The second shows HBM revenue becoming a large share of DRAM revenue. These are not mere HBM vocabulary; they are customer/capacity plus cash discipline and mix visibility.

### 4.2 Samsung: partial qualification and broad memory recovery must be split

Samsung is the main boundary case. On 2025-01-08 and 2025-01-31, the HBM-specific signal was still uncertain because Nvidia volume supply was not yet proven. However, the later 180D price path became strong because the broader memory cycle recovered. Therefore the rule should not hard-4C Samsung merely because HBM qualification lagged; it should cap C06 at Stage2-watch until volume supply is visible, while letting generic memory beta belong to C10 rather than C06.

### 4.3 SK Square: holding-company proxy is admissible only with discount

SK Square is not a memory producer, so it should never receive direct C06 credit. It can only enter C06 as a discounted proxy if three pieces join: SK hynix exposure, NAV-discount/value-realization route, and buyback/cancellation execution. That combination makes 2025-02-05 Actionable, but with explicit holding-company discount.
