## 3. Residual Mechanism

C21 is not “low PBR 금융주” in the abstract. The engine is ROE persistence plus capital return execution. A value-up headline is only the ignition key; the motor turns only when buyback, dividend, payout ratio, ROE durability, and asset-quality risk line up.

The main residual error is over-promoting label-only value-up rows. Large banks with visible payout execution can be Stage3-Yellow and later Green after reverify. Regional banks and brokerages need tighter guards because asset quality, cyclic trading income, or public-bank policy constraints can eat the rerating.
