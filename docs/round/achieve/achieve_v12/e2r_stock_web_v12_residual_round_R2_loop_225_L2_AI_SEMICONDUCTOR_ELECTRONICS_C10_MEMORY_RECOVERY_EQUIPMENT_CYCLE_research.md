## 5. Current calibrated profile stress test

- The current profile already blocks pure price-only blowoff, but C10 still needs a more precise ladder for `memory weather -> issuer route -> customer/order conversion -> shipment/utilization -> margin/cash`.
- `222800`, `095610`, `086390`, and `083310` show that a route can eventually work even after uncomfortable early MAE. Those should be high-MAE watch positives, not hard 4C.
- `166090`, `083450`, `089790`, and `092870` show the opposite side: product route or industry recovery alone should not receive the same Actionable/Yellow credit as direct order/shipment conversion.
- The repeated residual error is not “Stage2 is useful.” The sharper finding is that C10 needs an issuer-conversion freshness ladder; otherwise the model treats a weather report as if it were a purchase order.
