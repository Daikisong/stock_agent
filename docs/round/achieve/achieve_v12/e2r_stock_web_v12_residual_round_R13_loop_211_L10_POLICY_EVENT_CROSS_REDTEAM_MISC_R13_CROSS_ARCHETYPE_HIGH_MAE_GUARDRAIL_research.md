This loop does not add another sector-specific Cxx study. It converts the same recent sector evidence into a dedicated **R13 high-MAE geometry holdout**.

R13 is the correct scope because this is a cross-archetype guardrail, not a shipbuilding, construction, or consumer-export rule by itself.

Rule candidate:

```text
IF MAE90 <= -20% AND MFE90 < 15%
THEN treat as bad_high_mae_bridge_gap_false_positive unless later hard evidence repairs the bridge.

IF MAE90 <= -20% AND MFE90 >= 15%
THEN do not hard-block the archetype; apply staged-entry, position-size, or local 4B exit guard.

IF MFE90 >= 30% AND MAE90 > -12%
THEN preserve as control positive; generic high-MAE guard must not overblock similar hard-bridge rows.

IF a weak or 4C-like headline arrives after valuation reset AND MFE180 reopens materially
THEN route to Stage4C-Watch / false-break audit, not hard 4C.
```

The practical metaphor is a suspension bridge in wind. A deep drawdown is the bridge shaking. The question is whether the cables are still anchored: contract, channel, margin, repair, or reorder evidence. If the cables are gone, Stage2 should close. If the cables are intact, the right response is staged entry and exit geometry, not thesis demolition.

## 6. Interpretation

### 6.1 Bad high-MAE rows

The bad high-MAE cluster is defined by low MFE and deep MAE. 대양전기공업, CJ제일제당, 삼양식품 late export headline, and 풀무원 early record-sales row all show the same shape: the evidence was not empty, but the entry had too little upside oxygen before the drawdown hit. These should be capped at Stage2-Watch or no-promotion until the issuer-level money road becomes visible.

### 6.2 Surviving high-MAE rows

오리엔탈정공, HJ중공업, HD현대마린솔루션 reset row, and 농심 export-channel row had deep MAE but also material MFE. These are not clean Green rows. They are **surviving thesis with poor entry geometry**. The rule should not delete the archetype signal; it should attach staged-entry, smaller position, and local 4B exit guard.

### 6.3 Clean controls

동성화인텍 2025, 세진중공업, HJ중공업 C30, 계룡건설, 삼양식품 early, 농심 reset, and 동원개발 reset rows are control positives. They keep the guardrail honest: if the model simply punishes volatility, it will suppress exactly the kind of hard-bridge rows E2R is supposed to capture.

### 6.4 False-hard-4C audit

코오롱글로벌 and 동원개발 reset rows show why hard 4C needs a reset-price audit. A weak headline after valuation reset is not automatically a dead thesis. If MFE180 reopens with shallow MAE, the event should sit in Stage4C-Watch / false-break audit rather than hard 4C.
