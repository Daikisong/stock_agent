## 1. Scope
This is not a new sector-specific Cxx study. It is an R13 cross-archetype high-MAE holdout after the loop 203 Stage2 false-positive review. The purpose is to separate three patterns that the current calibrated profile can still blur:

```text
1. low-MFE / high-MAE entries where the thesis or bridge vocabulary was not enough
2. high-MFE / high-MAE entries where the thesis was real but the entry needed staged sizing or local 4B exit logic
3. clean positive controls where a hard bridge and low MAE should not be overblocked
```

## 5. Cross-Archetype Finding
High MAE is not one thing. It is a fork in the road. The same -30% drawdown can mean a broken Stage2 when upside never appears, or a sizing/timing problem when a hard commercial bridge later produces large MFE.

```text
low MFE + deep MAE
  -> bridge vocabulary or late headline was not enough
  -> downgrade to Stage2-Watch / No-Promotion / local 4B

high MFE + deep MAE
  -> thesis may be real but entry geometry is bad
  -> keep as staged-entry / position-size / local 4B exit guard

high MFE + shallow MAE
  -> hard bridge and conversion are already being recognized
  -> preserve Stage2-Actionable control; do not overblock
```
