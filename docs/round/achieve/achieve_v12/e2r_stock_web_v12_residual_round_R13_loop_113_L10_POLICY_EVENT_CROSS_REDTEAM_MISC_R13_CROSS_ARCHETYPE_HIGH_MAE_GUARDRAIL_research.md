## 13. Current Calibrated Profile Stress Test

Current profile behavior can be too permissive when a valid source event is allowed to persist as Stage2/Stage3 after the bridge fails. The R13 guardrail corrects the distinction:

```text
valid non-price event exists
  -> bridge not yet confirmed
  -> MAE90 <= -30% or MAE180 <= -35%
  -> cap at Stage2-Watch or force local Stage4B overlay
```

Hard Stage4C remains reserved for explicit non-price thesis break. HDC현대산업개발 and 후성 are therefore not treated as ordinary high-MAE volatility; they remain hard-break rows.
