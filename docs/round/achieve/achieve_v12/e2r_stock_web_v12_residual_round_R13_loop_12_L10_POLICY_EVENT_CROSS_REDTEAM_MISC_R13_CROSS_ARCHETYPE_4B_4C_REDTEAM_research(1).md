## 0. Validation scope

This file is not a live recommendation, not a current watchlist, not an auto-trading instruction, and not a `stock_agent` code patch. It is a standalone historical residual research Markdown file for the v12 post-calibrated runner.

The research question is narrow: after many local C01~C32 30-row / 50-row cleanup passes, which prior rows still look dangerous at the local 4B / full 4B / hard 4C boundary? The selected rows all have complete 30D/90D/180D MFE·MAE fields from prior stock-web-based v12 files, but most are `source_proxy_only` or reused price rows. Therefore this R13 checkpoint does not promote anything. It blocks promotion until profile/shard refetch and non-price bridge verification.

## 1. Selection rationale

`V12_Research_No_Repeat_Index.md` says R13 is not a normal sector round. It is for cross-archetype guardrail checks such as Stage2 false positives, 4B/4C red-team, accounting trust, and high-MAE protection. This run chooses `R13_CROSS_ARCHETYPE_4B_4C_REDTEAM` because the local ledger is now heavy with sector cleanup rows, and the remaining risk is not a new sector gap; it is a boundary problem.

The boundary problem is simple:

```text
Local 4B is useful as a watch bell.
Full 4B requires verified non-price bridge.
Hard 4C requires thesis break or destructive path evidence.
Source-proxy rows with deep MAE must not jump from local 4B to full 4B or Green.
```

## 6. Interpretation

The rows show the same pattern across battery, defense, content/IP, software/security, bio-healthcare, consumer export, and policy-adjacent cases. Price sometimes gave a strong MFE, but the same rows also carried very deep MAE. In a clean 4B promotion path, that is tolerable only when the non-price bridge has been refreshed: contract conversion, revenue/margin conversion, installed-base reorder, retention, CSM/reserve quality, PF balance-sheet repair, or direct policy-to-cash capture.

Without that bridge, the row is only a local watch signal. The profile should read it like a warning flare, not like a paved road. A local 4B marker can say “the market is paying attention”; it cannot say “the thesis is durable.”
