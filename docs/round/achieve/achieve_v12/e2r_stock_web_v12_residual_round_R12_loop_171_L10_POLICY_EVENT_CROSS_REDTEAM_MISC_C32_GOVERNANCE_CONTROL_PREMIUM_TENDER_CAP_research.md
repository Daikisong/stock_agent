## 17. Sector-Specific Rule Candidate
`L10_EVENT_PREMIUM_MINORITY_CAPTURE_AND_GATE_BREAK_GUARD_V1`: For L10 policy/event names, do not treat event premium as normal operating rerating. First check whether public minority holders can capture the premium, whether the price is below a credible event/tender value, and whether there is an unresolved vote/court/regulatory gate.

## 18. Canonical-Archetype Rule Candidate
`C32_TENDER_CAP_MINORITY_CAPTURE_AND_VOTE_GATE_GUARD_V1`: C32 Stage2-Actionable requires (1) public tender/control-premium event, (2) entry below plausible tender/control value, (3) minority-capture path, and (4) unresolved or positively shifting vote/court gate. Route to Stage4B when price trades near/above tender cap, when affiliate proxy value is derivative, or when private block premium cannot be captured. Route to Stage4C when shareholder vote, court ruling, or merger/control process breaks the event thesis.
