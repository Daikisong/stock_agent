
C15 cement rows should use a **shipment-adjusted spread ladder**:

```text
cement ASP / coal-cost headline
  -> issuer-level margin bridge
  -> shipment volume / utilization / inventory sanity
  -> working-capital or cash conversion
  -> price-phase sanity
```

해석은 간단하다. 시멘트 가격 인상과 유연탄 하락은 바람이다. 하지만 시멘트사는 고정비가 큰 장치산업이라 출하량이라는 물이 관을 지나가야 이익이 산다. 바람만 보고 돛을 올리면, 물이 빠진 항구에서 배를 띄우는 꼴이 된다.
