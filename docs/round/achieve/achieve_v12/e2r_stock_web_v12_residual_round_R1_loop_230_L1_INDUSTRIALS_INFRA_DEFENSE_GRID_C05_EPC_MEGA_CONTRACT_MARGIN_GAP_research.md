## 13. Current Calibrated Profile Stress Test

Current global axes helped prevent some raw policy/project headline overpromotion, but C05 still needs a more precise bridge. The profile is too optimistic when it treats offshore-wind award finality as if supplier cash conversion already arrived. It is too conservative when direct route plus post-reset asymmetry later produces large MFE. The residual rule therefore needs a ladder, not a flat penalty.

## 14. Stage2 / Yellow / Green Comparison

- Stage2-watch: broad offshore-wind award or generic project finality without listed issuer order.
- Stage2-Actionable: issuer product route plus project finality after price reset.
- Stage3-Yellow: issuer route has strong 90D/180D price validation and credible customer/project route.
- Stage3-Green: blocked in this loop unless direct order, margin, working-capital, and cash conversion are visible.

## 15. 4B Local vs Full-window Timing Audit

Counterexample rows are not hard 4C. They are 4B/watch because the route can survive while entry asymmetry is poor or attribution is indirect. `four_b_evidence_type` is source proxy / attribution cap / margin-cash lag, not price-only exit.

## 16. 4C Protection Audit

No hard 4C row is proposed. A supplier/proxy row with high MAE but a surviving project route should not be thesis death unless project, customer route, or issuer-level order/cash path is cancelled.

## 17. Sector-Specific Rule Candidate

`L1 offshore-wind supplier/project-finality rows should split project award existence, issuer-level supplier route, direct order/customer finality, margin bridge, working-capital/cash conversion, and price phase.`

## 18. Canonical-Archetype Rule Candidate

`C05 should add an offshore-wind supplier order-to-margin/cash phase ladder: project/policy finality -> issuer supplier route -> direct order/customer finality -> margin/cost proof -> working-capital/cash conversion -> phase sanity. Route-only after reset can be Actionable/Yellow; supplier proxy after rerating becomes 4B-watch; high-MAE with live route is not hard 4C.`
