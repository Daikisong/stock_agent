## 4. Residual finding

The common error is not that the initial evidence was fake. The common error is **stage persistence after the evidence bridge stayed unfinished**. These rows had legitimate initial signals—battery component contracts, nuclear-policy read-through, medical-device export/margin evidence—yet the path often delivered deep 90D/180D MAE before a second confirmation bridge appeared. The R13 rule should behave like a circuit breaker: it does not kill the thesis as hard 4C, but it refuses to let Stage2/Stage3 remain open without delivery, revenue recognition, margin, cash, shipment, utilization or retention confirmation.

## 5. Shadow rule candidate

```text
R13_HIGH_MAE_GUARDRAIL_REQUIRE_SECOND_BRIDGE_CONFIRMATION_OR_STAGE4B_OVERLAY_V130
```

Suggested behavior: if `MAE_90D_pct <= -25%` or `MAE_180D_pct <= -35%` while the row lacks second-bridge confirmation, cap the row at `Stage2-Watch` or add `Stage4B-LocalWatch`. Only route to hard 4C when non-price thesis break is explicit.
