핵심은 L5에서 “수출 증가, 글로벌 채널, record sales, 브랜드 턴어라운드, 면세/백화점 매출 회복, K-beauty/K-food popularity”라는 표지판만 보고 Stage2-Actionable을 열지 않는 것이다. 승격에는 최소 하나의 **reorder/channel repeat bridge**와 하나의 **listed-entity margin/OP/inventory conversion bridge**가 같이 필요하다.

```text
promotion-valid L5 money road
  = channel/reorder repeat
  + inventory or markdown quality
  + gross-margin or operating-profit conversion
  + listed-entity exposure purity
  + acceptable entry geometry

watch-only L5 row
  = export/channel/brand vocabulary
  + no repeated reorder or margin bridge
  + source proxy or missing URL
  + high-MAE or late growth headline

false-hard-4C audit
  = negative cost/marketing/inventory headline after valuation reset
  + forward MFE reopens
  + irreversible listed-entity break not proven
```
