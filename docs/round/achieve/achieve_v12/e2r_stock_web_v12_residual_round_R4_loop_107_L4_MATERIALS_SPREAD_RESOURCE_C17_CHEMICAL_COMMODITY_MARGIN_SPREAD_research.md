### Why C17 now

C17 is where the model can mistake a commodity word for a company-level earnings bridge. The weak path is:

```text
feedstock / petrochem / chemical spread headline -> price spike -> imagined OPM recovery
```

The stronger path is:

```text
realized product spread + utilization + inventory discipline -> OPM revision -> FCF conversion
```

The purpose of this pass is to preserve specialty-chemical positives while blocking generic spread-beta rallies from becoming Stage3-Green or full 4B without evidence.
