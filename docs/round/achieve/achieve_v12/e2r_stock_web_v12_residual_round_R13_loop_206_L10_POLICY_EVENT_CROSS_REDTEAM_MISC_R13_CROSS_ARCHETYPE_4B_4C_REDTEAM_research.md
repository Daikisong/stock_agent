## 5. Cross-Archetype Finding
4B and 4C should not be treated as the same warning light. 4B is the hot brake: the vehicle is still structurally intact, but event premium, late entry, price-only blowoff, or exhausted evidence makes the current entry dangerous. 4C is the broken axle: the value-capture path has actually failed through a vote/court gate, minority-capture, order/call-off break, qualification failure, or irreversible trust break.

```text
true 4B
  -> event cap, late-entry, price-only blowoff, or bridge exhaustion
  -> Stage4B-Watch / No-Promotion / re-entry only after fresh bridge

true 4C
  -> irreversible gate break or economics impairment
  -> hard Stage4C only after vote/court/order/trust confirmation

false hard 4C
  -> shock headline after valuation reset, but forward MFE reopens and bridge survives
  -> Stage4C-Watch / false-break audit, not immediate permanent block

high-MAE staged positive
  -> thesis and bridge are real, but entry geometry is bad
  -> staged-entry and position-size guard, not hard block

control positive
  -> named customer/contract/revenue/margin/retention bridge with strong MFE and tolerable MAE
  -> preserve Stage2-Actionable; avoid generic 4B/4C overblocking
```
