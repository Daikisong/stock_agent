## 1. Selection rationale
C31 is not a raw coverage shortage in the original No-Repeat Index; it is a quality-repair bucket with 118 representative rows. After this session already repaired many P0/P1 and R13 guardrail gaps, this pass targets a different C31 failure mode: **binding policy is not the same thing as listed-company money**.

Prior delivered C31 loop 172 focused on AI textbooks, telemedicine, KASA, KEPCO tariff and medical-school quota policy. This loop avoids that set. It also avoids the financial Value-up execution leaf and instead compresses five policy families: clean hydrogen power bidding, semiconductor support package, AI Basic Act, offshore wind special act, and virtual asset user-protection regulation.

The working analogy is simple: a policy announcement is a road sign. E2R Stage2-Actionable should wait for asphalt: award, contract, tariff, subsidy receipt, revenue timing, take-rate or margin/revision bridge.

## 6. Residual diagnosis
### 6.1 Clean hydrogen auction policy
The clean-hydrogen power auction opened a real policy market, but Doosan Fuel Cell and Bumhan Fuel Cell behaved like proxy baskets. The policy defined demand architecture; it did not, at the trigger date, name issuer-level awards, margins or revenue timing. Both rows had early event pop followed by severe 90D/180D MAE. C31 should keep these as Stage2-Watch or local 4B, not Stage2-Actionable.

### 6.2 Semiconductor support package
The semiconductor support package was binding and budget-backed. However, it still covered a broad ecosystem. DB HiTek had a tradable short-window MFE, but the row still needed an exit guard because the 180D drawdown was large. SFA Semiconductor had the same policy vocabulary with no issuer-specific utilization/revenue bridge and became a clearer false positive.

### 6.3 AI Basic Act
The AI Basic Act created a powerful policy-beta window for Saltlux and Konan Technology. The price path says the market did reward the policy family, but the later peak drawdown says the engine should not promote these directly to Stage3-Green. They are Stage2-Actionable only with event-cap / staged-entry / follow-through contract checks.

### 6.4 Offshore Wind Special Act
The Offshore Wind Special Act improved the licensing/permitting road. CS Wind and SK Oceanplant show why C31 needs issuer direct exposure. CS Wind had later MFE but immediate MAE was too deep, while SK Oceanplant gave a stronger long-window MFE because fabrication/offshore-plant exposure is closer to the permitted project money road.

### 6.5 Virtual Asset User Protection Act
The virtual asset law was a genuine market-structure event, but Woori Technology Investment is an indirect Dunamu/crypto beta proxy. MFE appeared, yet drawdown and beta dependence were too large for a clean policy-positive classification.

## 7. Current calibrated profile stress test
The current profile already blocks price-only blowoff and asks for non-price evidence. The residual error is subtler: C31 evidence can be non-price but still not be **company-money evidence**. A policy can be official, budgeted and legally binding while the listed issuer remains only a proxy.

Shadow rule candidate:
```text
C31_BINDING_POLICY_COMPANY_MONEY_BRIDGE_AND_PROXY_BETA_GATE_V2
```

Rule mechanics:
```text
if canonical_archetype_id == C31_POLICY_SUBSIDY_LEGISLATION_EVENT:
    require binding_policy_bridge = law_or_budget_or_formal_rollout_or_tariff_or_auction
    require issuer_money_bridge = named award/order/subsidy recipient/tariff revenue/take-rate/revenue timing/margin revision
    if binding_policy_bridge and issuer_money_bridge:
        Stage2-Actionable allowed
    elif binding_policy_bridge and no issuer_money_bridge:
        Stage2-Watch only
    if MAE90 <= -20 or drawdown_after_peak <= -35:
        attach local_4B_or_staged_entry_guard
    if policy is merely thematic or proxy-only:
        block Stage2-Actionable, mark C31 policy_proxy_false_positive
```
