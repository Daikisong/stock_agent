## 2. Research thesis

C02 should not be a loose “electricity keyword” bucket. It should reward only the cases where the power-infrastructure chain becomes measurable: named IDC/grid customer, busduct/transformer/cable/switchgear delivery route, backlog or capacity lock, and then revenue/margin conversion. Component words such as SMPS, ATS, MCCB, ESS or generic power electronics can be useful watch evidence, but without a named grid/datacenter order they behave like a cheap theme mask: they reflect the light but do not prove the engine is running.

The key split from this loop:

```text
Direct IDC/grid power-system supply + realized earnings bridge -> Stage2-Actionable / possible Yellow
Product identity or optionality without named order/backlog/margin bridge -> Stage2-watch / local 4B guard
```
