## 1. Selection memo

직전 loop 236은 L5 소비재 묶음인 C18/C19/C20을 **R13 Stage2 false-positive** 관점에서 재검증했다. 이번 loop는 같은 source coverage를 새로 늘리지 않고, 동일 L5 source rows를 **R13 high-MAE guardrail** 관점에서 다시 읽는다.

L5의 deep MAE는 항상 “틀림”이 아니다. 어떤 row는 K-food, K-beauty, apparel-OEM, retail 턴어라운드 thesis가 살아 있지만 진입 위치가 너무 뜨거워 MAE가 깊다. 반대로 어떤 row는 수출·브랜드·채널 vocabulary만 있고 reorder, inventory, margin bridge가 없어 MFE는 낮고 MAE만 깊다. 이 둘을 같은 칸에 넣으면 guardrail이 칼이 아니라 망치가 된다.

## 7. R13 rule candidate

```text
rule_candidate_id = R13_L5_HIGH_MAE_REORDER_INVENTORY_MARGIN_ENTRY_GEOMETRY_GATE_V2
production_scoring_changed = false
shadow_weight_only = true
```

핵심은 L5에서 `MAE90 <= -20%`를 무조건 실패로 보지 않고, **MFE와 listed-entity money road**를 같이 읽는 것이다.

```text
low MFE + deep MAE
  -> export/channel/brand/retail vocabulary는 있었지만 돈길이 약한 false positive
  -> Stage2-Watch, no-promotion, local 4B guard

high MFE + deep MAE
  -> thesis는 살아 있을 수 있지만 entry geometry가 나쁨
  -> staged-entry, position-size, local 4B exit guard

high MFE + shallow MAE
  -> reorder/channel repeat + inventory/margin/OP bridge와 가격경로가 같이 맞음
  -> Stage2-Actionable / Yellow control positive로 보존

source-proxy or missing URL row
  -> R13 watch/red-team row로만 사용
  -> promotion-valid evidence로는 단독 사용 금지
```

L5 소비재는 소비자의 발길과 계산대의 숫자가 다를 수 있다. 매장 앞 줄, 해외 채널 입점, SNS 인기, record sales headline은 모두 열기다. Stage2-Actionable로 올리려면 그 열기가 **반복 리오더, 재고 질, markdown discipline, 매출총이익률, 영업이익 전환**으로 가야 한다.
