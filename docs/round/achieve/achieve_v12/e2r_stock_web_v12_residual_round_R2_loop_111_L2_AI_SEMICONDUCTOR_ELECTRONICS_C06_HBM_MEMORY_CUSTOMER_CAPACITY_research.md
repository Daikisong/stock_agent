## 4. Interpretation

C06 should not treat every HBM-adjacent supplier as a direct customer-capacity lock. The positive rows show that memory test service, bonding/material, advanced film, and wet chemical proxies can work when the price path later confirms a revenue/margin bridge. The counterexample rows show the other half of the mechanism: OSAT and socket/connector labels can spike early, then lose more than 40% to 70% within the 180-trading-day window when direct customer lock or margin conversion is missing.

The shadow rule candidate is therefore not a global threshold change. It is a C06-specific bridge requirement:

```text
C06_verified_direct_HBM_customer_capacity_or_proxy_revenue_margin_bridge_required_before_Yellow_or_Green_plus_proxy_label_to_local_4B_watch_v2
```

Mechanically, C06 behaves like a power line with two transformers in series. Direct HBM customer capacity is the first transformer; revenue/margin/acceptance conversion is the second. If only the first rumor-voltage is present, the stock may flicker, but the rerating grid is not yet energized.
