## 13. Current Calibrated Profile Stress Test
The current calibrated profile already blocks generic price-only blowoff and requires non-price evidence for full 4B. This R13 loop does not repeat that global claim. It isolates the residual routing problem: some rows need **4B earlier**, some rows need **4C later**, and some verified bridge positives must survive the first sector-level disappointment.

Observed residual pattern:

```text
verified_bridge + shallow_MAE + large_MFE => keep Stage2/Yellow, add post-peak local 4B watch only after blowoff
label_or_proxy + low_MFE + deep_MAE => block Actionable/Yellow, route to local 4B
label_or_proxy + issuer-level non-price break + deep_MAE => hard 4C candidate
```

## 14. Stage2 / Yellow / Green Comparison
- Positive controls show that project bridge, reorder/sell-through bridge, spread-to-margin recovery, and ad operating leverage can keep Stage2/Yellow alive even when the same sector has many label-spike failures.
- Counterexamples show that commodity spread labels, brand/D2C labels, clinical labels, and IP-owner readthroughs can look actionable but produce low MFE90 and deep MAE90 when the bridge is absent.
- Stage3-Green strictness remains unchanged. This loop proposes a 4B/4C routing overlay, not a Green unlock.

## 16. 4C Protection Audit
Hard 4C should not fire on generic sector disappointment alone. For example, the C04/C15/C20/C26 positives would be wrongly killed if the model treated all legal-delay, material-spread, K-beauty, or ad-platform noise as thesis break. Conversely, C17/C19/C24/C27 counters show that once the issuer-level bridge is missing and the price path shows low MFE with deep MAE, local 4B or hard 4C must arrive before the model keeps calling the row actionable.
