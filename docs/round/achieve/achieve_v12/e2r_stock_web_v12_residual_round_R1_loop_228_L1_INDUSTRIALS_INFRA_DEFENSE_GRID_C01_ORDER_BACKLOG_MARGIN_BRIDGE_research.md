## 4. Residual diagnosis

C01 keeps failing when the model treats the first visible pipe as if cash is already flowing through it.  Nuclear, grid, rail, cable, and power-equipment route evidence can be real, but route existence is not the same as issuer-level order finality, and issuer-level order finality is still not the same as margin and cash conversion.  The usable distinction is a ladder:

```text
route evidence -> issuer/customer order finality -> margin/cost proof -> working-capital/cash conversion -> phase sanity
```

Positive rows in this loop show that after-reset route evidence plus direct customer/order or result bridge can justify Stage2-Actionable or Yellow; Hanwha Aerospace and Hyundai Rotem are strong Green-style C01 examples because their evidence is issuer/customer specific and their paths had controlled MAE.  Counterexamples show the opposite: route-only nuclear or cable exposure after a crowded rerating should become local 4B-watch, not Green, and high MAE alone should not become hard 4C when the issuer route survives.
