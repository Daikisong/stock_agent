## 3. Residual finding

L8에서 Stage2 오발은 대체로 같은 모양을 가진다.

1. C26은 search/ad/commerce/platform sales가 보이더라도 operating quality, net-income drag, trust/cost drag가 남아 있으면 MFE보다 MAE가 먼저 열린다.
2. C27은 global IP, demo, launch, user count, contract renewal만으로는 부족하다. liveops revenue, distributor economics, MD/reorder, platform take-rate가 붙어야 한다.
3. C28은 AI/security/cloud/PQC/zero-trust vocabulary만으로는 부족하다. recurring license, renewal, maintenance, managed-security run-rate, cloud/IDC utilization, margin conversion이 필요하다.

즉 L8의 Stage2는 “사람이 모였다”가 아니라 “반복 과금의 수도관이 열렸다”여야 한다. 손님이 가게 앞에 줄을 섰다는 사진만으로 매출총이익을 예약할 수 없는 것과 같다. 줄이 계산대로 흘러가고, 다시 방문하고, 고정비를 덮는 순간부터 Stage2-Actionable이 된다.
