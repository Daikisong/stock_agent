## 14. Stage2 / Yellow / Green Comparison

Stage2 can be given for product/process role plus public evidence. Stage2-Actionable requires named customer route, order/qualification, or HBM process bottleneck. Yellow requires conversion evidence after the first order: repeat PO, shipment, revenue bridge, or margin visibility. No row in this file justifies relaxing Green thresholds.

## 15. 4B Local vs Full-window Timing Audit

This loop mainly strengthens local 4B discipline. DI, STI, HPSP, NeoSem, LaserSel, and KCTech all show fast MFE or peak-headline paths followed by large MAE. The canonical-specific rule is not “sell every HBM equipment winner.” It is:

```text
If C07 evidence is only product identity, localization expectation, or one headline without repeat order/margin conversion, attach local 4B/profit-lock after fast reprice and block fresh Yellow/Green at peak.
```
