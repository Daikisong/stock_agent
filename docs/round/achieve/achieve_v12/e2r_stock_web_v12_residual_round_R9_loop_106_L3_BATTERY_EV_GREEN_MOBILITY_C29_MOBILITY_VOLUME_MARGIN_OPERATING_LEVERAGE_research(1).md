## 1. Selection rationale
V12는 순환 round가 아니라 coverage-index-first로 대상 canonical archetype을 고른다. 최신 no-repeat index의 static view에서는 C29가 rows=3 / need_to_30=27 / need_to_50=47인 Priority 0 저커버리지 축이다. 그러나 이 대화의 local ledger에서는 C29 loop 100~105가 이미 local 30-row floor를 만든 상태로 보이므로, 이번 loop는 C29를 static/operating 50-row band로 끌어올리는 cleanup pass로 둔다.

선택한 mechanism은 자동차/부품/타이어/열관리/물류가 같은 mobility label을 달고 움직일 때, 실제로는 회사별 volume → mix/ASP → OPM/FCF → capital return bridge가 없으면 Stage3-Green이나 full 4B로 올리기 어렵다는 잔여 오류다.

## 3. Research thesis
C29 is not a generic auto-beta bucket. It needs a company-level mechanism: volume growth, favorable mix/ASP, margin conversion, and preferably FCF or capital-return bridge. OEM leaders can become true positives when the volume/mix/capital-return bridge is visible. Parts, tire, thermal-management, logistics, and restructuring names often produce a spark of MFE, but without OPM/FCF confirmation the move behaves like a wet match: it flares, smokes, and then the MAE eats the signal.
