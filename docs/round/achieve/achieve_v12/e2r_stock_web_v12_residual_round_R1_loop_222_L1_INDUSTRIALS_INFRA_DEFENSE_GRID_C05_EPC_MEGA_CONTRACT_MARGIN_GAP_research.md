## 6. Current calibrated profile stress test

Current profile proxy: `e2r_2_2_rolling_calibrated`. This loop does not retest the old v2.0 baseline. It checks residual C05 errors after the already-applied bridge/watch/hard-4C axes.

Observed residual families:

1. **Order headline without cash conversion** — direct or semi-direct order/backlog evidence can justify Stage2/Yellow, but it should not borrow Green credit unless cost-rate, margin, and working-capital/cash conversion are visible.
2. **Result-quality Green trap** — quarterly results from high-value vessels are strong evidence when MAE is low, but a result row after or during a high-MAE phase needs 4B watch sizing.

4. **High-MAE live route is not hard 4C** — HJ중공업 and 한화오션 style rows can have live customer/order routes and still suffer drawdown; the right door is 4B/watch unless the route dies.
