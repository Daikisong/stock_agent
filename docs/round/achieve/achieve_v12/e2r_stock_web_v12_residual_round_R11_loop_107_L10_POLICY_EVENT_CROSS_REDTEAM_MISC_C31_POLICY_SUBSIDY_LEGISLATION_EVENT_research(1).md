## 1. Selection rationale

The static No-Repeat Index still places `C31_POLICY_SUBSIDY_LEGISLATION_EVENT` in Priority 0 with only 3 representative rows. Conversation-local summaries had already discussed C31, but a strict local parse found only a small number of complete C31 trigger rows with all required 30D/90D/180D MFE and MAE fields. This run therefore chooses C31 again, not to repeat a narrative, but to repair the machine-readable row surface.

The mechanism under test is simple: a policy, subsidy, legislation, tariff, tax, value-up, or accounting rule headline should not be promoted unless the rule becomes company-level cash, ROE, capital return, reserve/CSM quality, or subsidy capture. If the actual mechanism is ROE/PBR capital return, reroute to C21. If it is reserve/CSM/rate cycle, reroute to C22. C31 only owns the policy-to-company-cash bridge itself.
