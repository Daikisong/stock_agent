## 1. Scope / Why This Archetype

This pass returns to C30 after the R13 guardrail holdouts and the latest C01 component-backlog pass. C30 already has enough raw representative rows in the static No-Repeat Index, but it still has a dense residual-error surface: PF policy support, company-specific liquidity repair, earnings-loss headlines, valuation reset, and construction-sector proxy beta often wear the same hard hat while doing different jobs.

The loop deliberately avoids rematerializing the prior C30 symbols where possible. It adds a mid-cap builder / regional contractor leaf using HanShin, HJ Heavy Industries, Kolon Global, Dongwon Development, HS Hwasung, and Kyeryong Construction. The goal is not to make construction stocks bullish. The goal is to teach the stage machine that a construction shock must be routed through three different doors:

```text
company_specific_repair_bridge -> Stage2/Stage2-Actionable candidate
broad_PF_or_policy_relief_without_company_money_road -> Stage2-Watch / false-positive guard
loss_or_weak_earnings_after_valuation_reset -> Stage4C-Watch / false-hard-4C audit
```

## 6. Findings

### 6.1 C30 is not just a “PF crisis bucket”

C30 behaves like a building inspection, not a news filter. A building with a cracked facade can still have intact beams, while a freshly painted facade can hide a weak frame. HanShin's FY2023 row and Dongwon's 2024 PF-relief row both warn against promoting vague PF comfort into Stage2-Actionable. In both rows, the evidence was real, but the company-level money road was not strong enough.

### 6.2 Company-specific repair is the only clean promotion path

HJ Heavy Industries and HS Hwasung show the cleaner path. The repair was not just “construction sentiment improved.” There was visible cost-rate, earnings, or balance-quality evidence. The stock-web paths then opened large MFE with shallow MAE. These are the rows C30 should keep as Stage2-Actionable candidates.

### 6.3 Loss headlines after reset need a second gate

Kolon Global and Dongwon Development show why hard 4C needs a reset audit. The negative earnings headline was real, but after a depressed valuation base, forward MFE reopened. This does not turn loss-making builders into Green candidates. It simply says a hard 4C should require irreversible liquidity/equity/project failure, not only a single backward-looking loss headline.

### 6.4 Policy/proxy high-MFE needs staged entry, not automatic Green

Kyeryong's path produced very large MFE, but the trigger evidence can be contaminated by policy/capital-relocation beta rather than clean issuer-level order or margin conversion. The correct state is staged-entry with a local 4B exit guard. It should not be thrown away, but it also should not be compressed into clean C30 Green evidence.
