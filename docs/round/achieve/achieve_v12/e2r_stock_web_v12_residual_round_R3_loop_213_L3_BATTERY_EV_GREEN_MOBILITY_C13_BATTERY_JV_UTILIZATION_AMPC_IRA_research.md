## 1. Execution boundary
This artifact follows the v12 residual-calibration runner. It is not a live scan, not a stock recommendation, not an auto-trading instruction, and not a `stock_agent` code patch. It uses historical Stock-Web tradable raw OHLC rows to stress-test the already calibrated profile and propose only shadow, batch-handoff candidates.

## 2. Selection rationale
The No-Repeat Index says all C01~C32 canonical archetypes are already above the 80-row band, so the next unit of work is quality repair rather than row-count filling. Within Priority 1, C13 remains useful because AMPC/IRA, JV, and offtake headlines often look official but do not always turn into utilization, ex-credit margin, or cash conversion. This loop therefore avoids a generic “battery subsidy good” repeat and instead tests the missing links in the pipe.

```text
headline or MOU -> named route -> volume/call-off/utilization -> ex-credit margin/cash
Only the latter two links should unlock Yellow/Green; local-peak contract headlines and one-quarter losses need 4B/4C timing guards.
```
