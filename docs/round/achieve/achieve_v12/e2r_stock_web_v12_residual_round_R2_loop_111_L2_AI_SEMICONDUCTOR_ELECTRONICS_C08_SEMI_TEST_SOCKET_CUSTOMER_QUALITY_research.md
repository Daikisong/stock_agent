## 4. Core finding

C08 needs to distinguish **customer quality** from **customer-quality vocabulary**. A test socket, probe-card PCB, final-test tool, OSAT expansion, or packaging roadmap can be a valid Stage2 signal, but the conversion chain must pass through qualification, reorder, shipment, revenue recognition, and margin bridge before Stage3-Yellow/Green is allowed.

```text
test/socket/probe-card/OSAT/customer-quality headline
  -> customer qualification or named evaluation
  -> repeat order / reorder or mass-production pull-through
  -> shipment and revenue recognition
  -> margin bridge and low-MAE path
  -> C08 Stage3 eligibility
```

Leeno, Tiger Elec, and Hana Micron are positive-with-guardrail rows: the value-chain signal was real, but all three still needed local 4B exits after peak. ISC, Nepes, Doosan Tesna, Protech, and GigaVis show the failure mode: development, final test, OSAT strategy, or small named order language can create MFE, but without reorder/revenue/margin bridge the 90D/180D path is a high-MAE trap.
