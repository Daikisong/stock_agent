### Why C16 now

C16 is the archetype where the market often mistakes scarcity language for cash conversion. The weak path is:

```text
resource / policy / strategic supply keyword -> price spike -> imagined scarcity premium
```

The durable path is:

```text
funded offtake / customer capacity lock / regulated project / order book -> volume or ASP -> margin -> working-capital absorption -> FCF
```

This pass keeps the second path available while making the first path Stage2/local-4B only until verified.
