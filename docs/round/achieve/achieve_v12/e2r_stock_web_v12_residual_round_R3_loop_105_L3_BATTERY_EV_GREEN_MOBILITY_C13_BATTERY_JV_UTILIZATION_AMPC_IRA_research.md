## 1. Selection rationale

`V12_Research_No_Repeat_Index.md` 기준 C13은 static table에서 `rows=15 / need_to_30=15 / need_to_50=35`인 Priority 0 잔여 축이다. 이 세션의 local ledger에서는 C13 loop 100/102/103/104를 통해 30-row floor에 도달했으나, Priority 1 운영권인 50-row band에는 아직 부족하다. 따라서 이번 loop는 C13을 30에서 50까지 끌어올리는 cleanup pass로 잡았다.
