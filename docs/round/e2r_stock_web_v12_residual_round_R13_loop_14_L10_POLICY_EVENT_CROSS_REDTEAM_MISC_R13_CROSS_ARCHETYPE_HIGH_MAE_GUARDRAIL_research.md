## 14. Stage2 / Yellow / Green Comparison

R13 does not re-prove that Stage2 is earlier than Green. The new residual is narrower: if a row has MAE_90D <= -30% and the evidence is proxy/theme-only, Green promotion should be blocked even if the total score clears a threshold. If the same high MAE is followed by contract/customer/revision recovery evidence, the row can remain Stage2-Actionable or Stage3-Yellow, but not automatically Green.

## 15. 4B Local vs Full-window Timing Audit

The R13 sample preserves local/full 4B split where source rows provided it. The rule is kept but refined:

```text
existing_axis_tested = full_4b_requires_non_price_evidence
existing_axis_strengthened = true
new_axis_proposed = risk_only_4B_watch_before_full_4B
```

Price-only local peaks are not full 4B. However, when valuation or event premium is already stretched and MAE after the 4B row is > 40% in magnitude, the system should allow a risk-only 4B watch tier. This is a smoke alarm, not a demolition order.

## 16. 4C Protection Audit

The 4C rows show that hard 4C routing is conceptually correct but often late. The proposed shadow rule is not “turn every rumor into 4C.” It is: when legal/regulatory/control evidence breaks the original bridge, open an early 4C watch before the final official failure.
